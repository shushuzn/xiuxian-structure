# xiuxian-structure

基于《凡人修仙传》参考的修仙世界观架构。

从底层开始，逐层搭建。

## 架构

```
xiuxian-structure/
├── 索引.md              # 各体系关联关系
├── 境界体系/            # 境界
│   ├── 境界.md          # 定义框架
│   ├── 炼气期.md         # 修仙第一步
│   └── 筑基期.md         # 道基初成
├── 灵根体系/            # 先天资质
│   └── 灵根.md           # 品级、属性、检测方式
├── 天地灵气/            # 世界资源
│   └── 灵气.md           # 浓度、灵脉、属性、界面差异
├── 功法体系/            # 修炼方法
│   └── 功法.md            # 品级、属性、修炼方式、传承
├── 丹药体系/            # 辅助外物
│   └── 丹药.md            # 品级、药力、丹方、炼丹条件
└── 法器体系/            # 战斗器物
    └── 法器.md            # 品级、属性、驱动条件、认主方式
```
