# 修仙体系关系图

本目录包含从 `data/relations.yaml` 自动/手工绘制的 mermaid 图。

## 境界晋升图

```mermaid
flowchart TD
    LQ[炼气期<br/>120年] -->|筑基丹| ZJ[筑基期<br/>200年]
    ZJ -->|结丹丹| JD[结丹期<br/>500年]
    JD -->|培婴丹| YY[元婴期<br/>1000年]
    YY -->|化神辅助| HS[化神期<br/>2000年]
    HS --> LX[炼虚期<br/>5000年]
    LX --> HT[合体期<br/>10000年]
    HT --> DJ[渡劫期]
    DJ --> DC[大乘期]

    style LQ fill:#e3f2fd
    style ZJ fill:#bbdefb
    style JD fill:#90caf9
    style YY fill:#64b5f6
    style HS fill:#42a5f5
    style LX fill:#2196f3
    style HT fill:#1e88e5
    style DJ fill:#ff9800
    style DC fill:#f57c00
```

## 灵脉等级-势力规模

```mermaid
flowchart LR
    WX[微型灵脉<br/>5km] --> XX[小型灵脉<br/>50km]
    XX --> ZX[中型灵脉<br/>100km]
    ZX --> DX[大型灵脉<br/>500km]
    DX --> CJ[超级灵脉<br/>1000km]

    WX -.-> SX[散修]
    XX -.-> XS[小型宗门]
    ZX -.-> ZS[中型宗门]
    DX -.-> DS[大型宗门]
    CJ -.-> TS[顶级宗门]
```

> 提示：GitHub / VS Code / Typora 等支持 mermaid 的渲染器会直接显示图。
> 渲染失败时检查 `data/relations.yaml` 中的 id 是否对应 `data/*.yaml` 中实际存在的 id。
