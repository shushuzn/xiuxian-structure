# 概念汇总（全局术语表）

> 修仙体系所有核心概念的统一术语表。
> 每个概念在项目中被多处引用，本表保证术语一致、定义清晰。
> 当不同文档使用同一术语时，以本表为准。

## 基础概念

### 修仙者 / 修士

- **定义**：修炼灵气、追求长生的人或妖
- **分类**：人修（人族修士）/ 妖修（妖兽修炼者，参考 [妖修](../妖修体系/妖修.md)）
- **境界**：参考 [境界](../境界体系/境界.md)（9 大境界 + 飞升）

### 灵气

- **定义**：天地间弥漫的灵力能量，是修仙的根本资源
- **分类**（按浓度）：稀薄 / 普通 / 浓郁 / 化液
- **分类**（按形态）：普通灵气 / 仙气（仙界）/ 神气（神界，传说级）
- **详见**：[天地灵气](../天地灵气/灵气.md)

### 境界

- **定义**：修仙者实力等级划分，9 大境界代表生命层次跃迁
- **顺序**：炼气 → 筑基 → 结丹 → 元婴 → 化神 → 炼虚 → 合体 → 渡劫 → 大乘
- **飞升后**：进入 [仙界](../飞升体系/仙界.md)（meta-mechanism，不增加境界）
- **详见**：[境界](../境界体系/境界.md)

## 资质 / 修炼资源

### 灵根

- **定义**：修仙者的先天资质，决定能否感应灵气及修炼效率
- **品级**（从低到高）：废灵根 / 伪灵根 / 真灵根 / 天灵根 / 变异灵根
- **属性**：金 / 木 / 水 / 火 / 土（5 行）+ 雷 / 冰 / 风（3 变异）
- **详见**：[灵根](../灵根体系/灵根.md)

### 功法

- **定义**：修仙者引导灵气在经脉中运转、提升修为的方法体系
- **品级**（从低到高）：黄级 / 玄级 / 地级 / 天级（下界）/ 仙级 / 神级（仙界）
- **详见**：[功法](../功法体系/功法.md)

### 灵石

- **定义**：蕴含灵气的天然矿石，是通用货币与能量来源
- **品级**（下界）：下品 / 中品 / 上品 / 极品
- **仙界**：仙灵石（5 档，参考 [仙灵石](../飞升体系/仙灵石.md)）
- **详见**：[灵石](../灵石体系/灵石.md)

## 战斗 / 辅助

### 法器（品级递进）

| 品级 | 等级 | 蕴含器灵 | 驱动境界 |
|---|---|---|---|
| 低阶法器 | 1 | 无 | 炼气期 |
| 中阶法器 | 2 | 无 | 筑基期 |
| 高阶法器 | 3 | 无 | 结丹期 |
| **法宝** | 4 | 雏形 | 元婴期 |
| **灵宝** | 5 | 完整 | 化神期 |
| **传说级** | 6 | 大成 | 化神+ |
| **仙器** | 7 | 至臻 | 仙界 |

- **详见**：[法器](../法器体系/法器.md) · [器灵](../器灵体系/器灵.md)

### 阵法

- **定义**：以阵旗、阵盘、灵石为媒介的法术体系
- **类型**：攻击 / 防御 / 困敌 / 聚灵 / 幻术 / 辅助
- **等级**（从低到高）：凡阵 / 灵阵 / 宝阵 / 仙阵
- **特殊**：渡劫阵（突破辅助）、护山大阵（宗门必备）
- **详见**：[阵法](../阵法体系/阵法.md)

### 符箓

- **定义**：绘制于符纸上的法术载体，激发后释放预封法术
- **等级**（从低到高）：凡符 / 灵符 / 宝符 / 仙符
- **类型**：攻击符 / 防御符 / 遁符 / 封印符 / 传讯符 / 辅助符
- **详见**：[符箓](../符箓体系/符箓.md)

### 丹药

- **定义**：以灵药为原料、经炼丹师炼制的辅助修仙之物
- **品级**（从低到高）：凡品 / 灵品 / 宝品 / 仙品
- **类型**（按用途）：突破类 / 增益类 / 恢复类 / 辟谷类 / 抗心魔类
- **详见**：[丹药](../丹药体系/丹药.md)

## 生命层次

### 神识（等级递进）

| 等级 | 境界 | 范围 | 能力 |
|---|---|---|---|
| 微神识 | 炼气期 | 3-10 丈 | 模糊感知 |
| 小神识 | 筑基期 | 30-100 丈 | 可画符 / 炼丹 |
| 大神识 | 结丹期 | 100-500 丈 | 穿墙 / 附身低阶 |
| **元神识** | 元婴期+ | 数里-数十里 | 离体 / 夺舍 / 识战 |

- **详见**：[神识](../神识体系/神识.md) · [神识攻击](../神识体系/神识攻击.md)（识刺 / 识爆 / 识域 / 识灭）

### 元婴

- **定义**：神魂具象化后的具形化身（[元婴期](../境界体系/元婴期.md)修士的核心）
- **能力**：可离体存在、附身、夺舍
- **风险**：雷劫中元婴是软肋（被劈散则形神俱灭）

## 突破 / 考验

### 心魔

- **定义**：突破大境界或闭关时，神识深处因执念、贪嗔痴等情绪而生的"心障"
- **类型**（按佛教三毒 + 执念）：贪魔 / 嗔魔 / 痴魔 / 执念魔
- **渡过工具**：[清心丹](../丹药体系/清心丹.md) · [凝神丹](../丹药体系/凝神丹.md) · 心境磨练
- **详见**：[心魔](../心魔体系/心魔.md)

### 雷劫（品级递进）

| 雷劫 | 道数 | 强度 | 突破 |
|---|---|---|---|
| 三九天劫 | 9 道 | 中阶 | 结丹 → 元婴 |
| 六九天劫 | 18 道 | 高阶 | 元婴/化神 → 化神/炼虚 |
| 九九天劫 | 27 道 | 至高阶 | 炼虚 → 合体 → 渡劫 → 大乘 |
| **飞升之劫** | +3 道 | 极巅 | 大乘 → 飞升 |

- **详见**：[雷劫](../雷劫体系/雷劫.md) · [飞升之劫](../飞升体系/飞升之劫.md)

### 仙劫（飞升后）

- **类型**：雷仙劫 / 火仙劫 / 风仙劫 / 心魔仙劫 / 复合仙劫
- **频率**：5000 - 50000 年一次
- **失败后果**：道消（消散于天地）
- **详见**：[仙劫](../飞升体系/仙劫.md)

## 势力 / 世界

### 势力

- **定义**：修仙界各种组织形态的总称
- **类型**：宗门 / 家族 / 散修联盟 / 王朝 / 妖修 / 海盗
- **规模**：散修 / 小型 / 中型 / 大型 / 顶级
- **详见**：[势力](../势力体系/势力.md)

### 妖兽 / 妖修

- **妖兽**：天地灵气滋养的兽类，开启灵智后可修炼
- **妖修**：妖兽修炼化形后的修仙者
- **品级**：凡兽 / 灵兽 / 妖兽 / 大妖 / 妖王 / 妖皇 / 妖帝
- **详见**：[妖兽](../妖兽体系/妖兽.md) · [妖修](../妖修体系/妖修.md)

### 秘境

- **定义**：独立于主世界之外的小世界或封闭区域
- **类型**：浮岛秘境 / 地下秘境 / 空中秘境 / 空间秘境
- **详见**：[秘境](../秘境体系/秘境.md)

## 飞升 / 仙界

### 飞升

- **定义**：大乘期修士渡过飞升之劫，进入仙界的过程
- **三阶段**：飞升之劫 → 飞升过程 → 仙界落地
- **成功率**：约 10%
- **不可逆**：飞升后几乎无法返回下界
- **详见**：[飞升](../飞升体系/飞升.md)

### 仙界

- **定义**：飞升后的新界面，与下界规则完全不同
- **关键差异**：仙气浓郁 / 寿元无限 / 法则完全显现 / 不可干预下界
- **详见**：[仙界](../飞升体系/仙界.md)

### 仙职

- **等级**（从低到高）：散仙 → 仙徒 → 仙使 → 仙官 → 仙将 → 仙君 → 仙王 → 仙帝
- **详见**：[仙职](../飞升体系/仙职.md)

### 仙灵石

- **品级**（从低到高）：下品 → 中品 → 上品 → 极品 → 仙晶
- **能量密度**：仙灵石 ≈ 10000 倍上品灵石
- **详见**：[仙灵石](../飞升体系/仙灵石.md)

### 界面壁障

- **定义**：不同界面（人界/灵界/仙界/魔界）之间的天然屏障
- **类型**：单向壁障（飞升）/ 双向壁障（跨界）/ 封印壁障
- **详见**：[界面壁障](../飞升体系/界面壁障.md)
