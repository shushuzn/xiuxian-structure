  ### 4 类契约深度对比

  | 契约类型 | 关系强度 | 自由度 | 协同度 | 代价 | 适用对象 |
  |---|---|---|---|---|---|
  | 主从契约 | 中 | 灵兽受限 | 低 | 灵兽常反抗，易激发嗔魔 | 灵兽 / 妖兽 |
  | 平等契约 | 弱-中 | 双方自由 | 中 | 关系易破裂 | 灵兽 / 妖修 / 人 / 势力 |

  | 魂契 | 最强（极端） | 双方深度绑定 | 超越灵魂 | 修为倒退 1-3 小境界 + 折寿 + 禁忌 | 道侣 / 至亲 |

  ### 5 档可能的结局

  | 路径 | 关系终态 | 风险 | 长期收益 |
  |---|---|---|---|
  | 灵魂契约 | 本命绑定 | 主人死它重伤 | 心意相通，最终化形期妖修 |
  | 平等+长期培养 | 平等 → 准魂契 | 极低 | 数十年后升级为「灵魂至交」 |
  | 平等+保持距离 | 平等朋友 | 极低 | 各自修行，千年友谊 |
  | 主从+强行压制 | 主从（高怨恨） | 30-50% 反噬 | 灵兽完全反噬，两败俱伤 |
  | 放归山林 | 无契约 | 0 | 错失灵兽，保全自由（它偶尔回报） |

  ### 实际案例

  - **aoyue + 九幽寒蟒**：灵魂契约 → 渡劫时灵兽挡一道天雷（最终牺牲）
  - **黑岩海盗 + 海妖**：主从契约 → 海妖消极怠工，常有逃脱
  - **化形期妖修 + 人类修士**：平等契约 → 双方自由，类似道友

## 设计说明

### 节点结构

- **幼狼抉择**（scene 节点）— 单节点演示 4 类契约 + 5 档结局
- 采用「文档化多结局」格式：所有可能的选择都在 text 中以表格形式呈现
- 玩家实际游玩时，可基于此节点做选择，由其他剧本驱动跳转

### 关联的 v1.7 体系

| 体系 | 在本节点的体现 |
|---|---|
| [契约](../../契约体系/契约.md) | 4 种契约方式 |
| [灵魂契约](../../契约体系/灵魂契约.md) | 本命契约（aoyue 模式） |
| [平等契约](../../契约体系/平等契约.md) | 双方自由，可升级为准魂契 |
| [主从契约](../../契约体系/主从契约.md) | 高反噬风险（灵兽袋基础） |
| [魂契](../../契约体系/魂契.md) | 平等契约的终极升级（禁忌） |
| [心魔体系 - 嗔魔](../../心魔体系/嗔魔.md) | 主从契约激发嗔魔 |

### 数据字段

- `契约方式` — 当前契约（灵魂/平等/主从/未定）
- `灵兽态度` — 信任/服从/怨恨
- `灵兽品级` — 凡品/灵品/宝品/仙品
- `灵兽服从度` — 0-100，影响战斗配合

### 复用方式

将「幼狼抉择」节点嵌入玩家遇到灵兽的场景，让玩家在 text 中查看所有契约方式
与对应的风险/收益，作为灵兽驯养的入门教学。
