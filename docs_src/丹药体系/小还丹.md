# 小还丹

## 定位
凡品恢复类丹药，恢复法力、治疗轻伤。
炼气期修士最常用的"回血回蓝"丹药，散修出门必备。

## 等级
- **品级**：凡品
- **用途**：恢复类
- **适用境界**：[炼气期](../境界体系/炼气期.md)

## 功效
- **法力恢复**：短时间内恢复约三成法力
- **轻伤治疗**：治疗皮肉伤、轻微经脉损伤
- **神识舒缓**：缓解神识疲劳

## 炼制
- **难度**：低
- **主药**：十年灵药（"还元三味"：还元草、续断、活血藤）
- **辅药**：普通草药即可
- **炼丹师**：炼气期即可炼制
- **丹炉**：普通丹炉
- **火焰**：凡火即可

## 稀有度
- **稀有度**：常见
- 主药在野外山林中常见
- 几乎所有宗门都有储备

## 限制
- 仅对炼气期有效，筑基以上服用效果甚微
- 无法治疗重伤或经脉重创
- 多次服用会产生抗药性
