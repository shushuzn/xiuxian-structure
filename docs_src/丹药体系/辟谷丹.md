# 辟谷丹

## 定位
凡品辟谷类丹药，替代进食、维持生机。
闭关修炼、外出探险、辟谷修炼时的必备之物，一颗可支撑数日不食。

## 等级
- **品级**：凡品
- **用途**：辟谷类
- **适用境界**：[炼气期](../境界体系/炼气期.md)

## 功效
- **替代进食**：一颗可支撑炼气期修士 7-10 日不食
- **维持生机**：保持体力充沛，不影响修炼
- **洁净肠胃**：长期服用有轻微洗筋伐髓之效

## 炼制
- **难度**：低
- **主药**：五谷灵草、灵泉水
- **辅药**：普通谷物即可
- **炼丹师**：炼气初期即可炼制
- **丹炉**：普通丹炉
- **火焰**：凡火即可

## 稀有度
- **稀有度**：常见
- 几乎所有宗门与散修都会自备
- 价格低廉，常以颗为单位交易

## 限制
- 仅对低阶修士有效；高阶修士本身已可辟谷
- 长期仅靠辟谷丹会导致气血不畅，需偶尔进食调和
- 不能替代真正的修炼丹药
