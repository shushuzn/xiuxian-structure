# 黄龙丹

## 定位
灵品增益类丹药，提升修为、加速修炼。
筑基修士日常修炼的辅助丹药之一，长期服用可显著缩短修炼周期。

## 等级
- **品级**：灵品
- **用途**：增益类
- **适用境界**：[筑基期](../境界体系/筑基期.md)

## 功效
- **修为提升**：单颗可提升相当于月余打坐的修为
- **加速修炼**：长期服用可使修炼速度提升 20-30%
- **法力精纯**：略微纯化法力，提升道基稳固度

## 炼制
- **难度**：中
- **主药**：百年灵药（"黄龙三味"：黄龙参、龙涎草、金线藤）
- **辅药**：数十年灵药若干
- **炼丹师**：筑基期即可炼制
- **丹炉**：中品灵炉
- **火焰**：灵火即可

## 稀有度
- **稀有度**：常见
- 主药在大型灵脉附近偶有产出
- 中型宗门可批量采购

## 限制
- 仅对筑基期有效，高阶修士服用无益
- 不可替代突破丹药，仅作日常辅助
- 长期服用易产生抗药性，需搭配其他丹药轮换
