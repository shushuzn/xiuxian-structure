# 冥器

## 定义

冥界高阶法器——比仙器更阴寒、比神器更死寂，蕴含完整的冥道法则。
冥器是鬼修施展冥功的媒介，没有冥器的鬼修如同无牙之虎。
冥器由冥界冥匠炼制，部分冥器为冥道自然孕育。

## 等级（6 档）

| 等级 | 来源 | 驱动 |
|---|---|---|
| 凡冥器 | 冥匠炼制 | 游魂 |
| 灵冥器 | 冥匠精炼 | 鬼徒 - 鬼使 |
| 真冥器 | 鬼尊炼制 | 鬼官 - 鬼将 |
| 大冥器 | 冥帝炼制 | 鬼君 - 鬼王 |
| 至冥器 | 冥道孕育 | 冥帝 |
| 道器 | 传说级 | 冥祖（不可测）|

## 特性

### 蕴含完整冥道法则

- 冥器蕴含完整的冥道法则
- 使用者可直接调用冥道之力
- 不需要像仙器那样「沟通」器灵

### 与冥格绑定

- 高阶冥器需特定冥格才能驱动
- 冥格不足者使用高阶冥器会魂飞魄散

### 不可复制

- 高阶冥器是唯一的
- 同款冥器不可复制
- 每件冥器都有独特「冥痕」

## 代表冥器（传说级）

### 判官笔

- **等级**：至冥器
- **能力**：判定生死、书写轮回
- **持有者**：历代判官

### 生死簿

- **等级**：至冥器
- **能力**：记录万物生死
- **持有者**：冥帝

### 轮回盘

- **等级**：至冥器
- **能力**：运转六道轮回
- **持有者**：冥帝

### 孟婆汤勺

- **等级**：至冥器
- **能力**：赐予忘川之水（让亡灵忘却前生）
- **持有者**：孟婆

### 幽冥剑

- **等级**：至冥器
- **能力**：杀伐第一冥器
- **持有者**：冥帝

### 万鬼令

- **等级**：至冥器
- **能力**：召唤万鬼
- **持有者**：酆都大帝

## 炼制

### 原料

- 冥石
- 死气精华
- 冥兽骨骼 / 鳞甲
- 冥道碎片
- 亡灵魂魄（顶级原料）

### 冥匠

- 普通冥匠 — 炼制凡冥器
- 鬼君级冥匠 — 炼制真冥器
- 鬼尊级冥匠 — 炼制大冥器
- 传说级冥匠 — 至冥器

## 备注

- 冥器炼制需大量冥气
- 冥器比神器更重审判
- 至冥器共 6 件，对应六大冥职
