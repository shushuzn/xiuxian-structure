# 器灵

## 定义

法器内部**孕育的意识体**——是法器的"灵魂"，让法器从死物变成可与主人沟通、甚至部分自主行动的"活物"。
器灵并非天生，而是经长期温养、天材地宝滋养、主人渡劫淬炼等机缘下逐渐诞生。
器灵是法宝、灵宝以上法器的核心标志，是法器体系中的最高阶形态。

## 构成要素

### 本质

- **法器的灵魂** — 器灵依托法器存在，无法独立生存
- **意识体** — 拥有自己的感知、记忆、情感（虽不如人完整）
- **可成长** — 随时间、滋养、战斗积累而逐渐强大
- **有性格** — 不同法器孕育的器灵性格迥异（受炼制者、原材料、机缘影响）

### 起源（按概率排序）

- **自然孕育** — 法器长期存放于灵脉之上，经数千年自然温养，可能诞生器灵（概率极低）
- **天材地宝催化** — 在炼制时加入特殊灵物（如万年灵木、化形期妖丹），可大幅提高器灵诞生概率
- **主人渡劫淬炼** — 主人渡劫时法器一同受劫雷淬炼，劫雷可激发器灵（参考 [雷劫淬体](../雷劫体系/雷劫淬体.md)）
- **战斗觉醒** — 法器在生死之战中被激发潜能，可能诞生器灵
- **血祭邪法** — 以修士精血或魂魄强行灌注，催生器灵（多导致器灵反噬）

### 等级

| 等级 | 对应法器品级 | 意识程度 | 能力 |
|---|---|---|---|
| 雏形 | 法宝 | 微弱意识 | 仅能传信、警示 |
| 小成 | 法宝-灵宝之间 | 较完整意识 | 可沟通、辅助操控 |
| 完整 | 灵宝 | 完整意识 | 可沟通、部分自主行动 |
| 大成 | 传说级（化神+） | 与主人心意相通 | 可独立战斗、自行护主 |
| 至臻 | 仙器（飞升期） | 完美融合 | 器灵即法器，法器即器灵 |

### 与主人关系

- **自愿认主** — 器灵认可主人后自愿结契，最稳定（参考 [器灵认主](器灵认主.md)）
- **强迫认主** — 主人强行压制器灵意识，关系紧张，易反噬
- **平等共生** — 大成器灵与主人平等相待，类似道友关系
- **反噬** — 器灵吞噬主人神魂，夺取法器控制权（参考 [器灵反噬](器灵反噬.md)）

### 能力（按等级递进）

- **传信** — 雏形即可，向主人传递法器感知到的信息
- **警示** — 危险来临时的自动预警
- **沟通** — 与主人通过神识交流
- **辅助操控** — 在战斗中自动调整攻击轨迹、防御角度
- **自主行动** — 主人神识离体时，器灵可独立操控法器护主
- **独立战斗** — 大成器灵可作为独立战力，与主人配合作战
- **自主修炼** — 至臻器灵可自主吸纳灵气修炼，反哺主人
