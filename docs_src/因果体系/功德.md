# 功德

## 定义

**功德**是善业的表现，是积累福报、提升气运的根本。修士通过行善、布施、护生、度化等行为积累功德。

## 来源

### 1. 布施行善
- 救助凡人
- 资助散修
- 赈济灾荒

### 2. 护生度化
- 保护弱小
- 化解纷争
- 救渡亡魂

### 3. 弘扬正道
- 传播正道
- 培养弟子
- 著作立说

### 4. 斩妖除魔
- 击杀恶妖
- 铲除邪修
- 护一方平安

## 功德等级

| 等级 | 描述 | 效果 |
|---|---|---|
| 小功德 | 单一善行 | 一年好运 |
| 中功德 | 持续善行 | 十年好运 |
| 大功德 | 重大善行 | 百年好运 |
| 圆满功德 | 累积无量大功德 | 飞升契机 |

## 功德的作用

### 1. 修炼
- 渡劫时减轻天罚
- 突破时增加成功率

### 2. 飞升
- 大功德者可顺利飞升
- 圆满功德者得天道认可

### 3. 气运
- 提升气运
- 增加奇遇概率
- 危难时遇贵人

### 4. 转世
- 转生善道（天道/人道）
- 投胎富贵之家

## 功德与业力

- **正相关**：行善增功德
- **负相关**：作恶增业力
- 两者在**因果天平**上抵消
