# 因果业报

## 定义

**因果业报**是业力的最终兑现——通过**轮回**、**天劫**、**现世报**等机制显现。

## 业报类型

### 1. 现世报
- **时间**：今生
- **方式**：直接显现
- **例**：害人者反害己

### 2. 来世报
- **时间**：死后转世
- **方式**：转生善道/恶道
- **例**：善人转生富贵，恶人转生贫贱

### 3. 累世报
- **时间**：多世累加
- **方式**：累世因果交织
- **例**：某一世享福，源自前世积德

### 4. 死后清算
- **时间**：死后
- **方式**：阎王审判
- **例**：冥界判定功过

## 善报

### 1. 长寿
- 善人往往长寿

### 2. 健康
- 善人少病

### 3. 富足
- 善人常得财富

### 4. 子女贤孝
- 善人后代昌盛

### 5. 死后善道
- 转生天道/人道

## 恶报

### 1. 短命
- 恶人往往短命

### 2. 多病
- 恶人多灾多病

### 3. 贫贱
- 恶人常处贫贱

### 4. 不孝子孙
- 恶人后代不肖

### 5. 死后恶道
- 转生饿鬼/地狱

## 业报的轻重

### 1. 看动机
- 善心作善事 → 大善报
- 恶心作善事 → 小善报
- 善心作恶事 → 小恶报
- 恶心作恶事 → 大恶报

### 2. 看对象
- 对父母不孝 → 大恶报
- 对众生平等 → 大善报

### 3. 看影响
- 影响广者业重
- 影响微者业轻
