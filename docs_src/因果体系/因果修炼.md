# 因果修炼

## 定义

**因果修炼**是通过**积累功德、消减业障**的方式提升气运、福报与修行境界的修炼法门。

## 修炼原则

### 1. 诸恶莫作
- 不杀生
- 不偷盗
- 不邪淫
- 不妄语
- 不两舌
- 不恶口
- 不绮语
- 不贪
- 不嗔
- 不痴

### 2. 众善奉行
- 布施
- 持戒
- 忍辱
- 精进
- 禅定
- 智慧

### 3. 自净其意
- 时时观照自心
- 断除贪嗔痴

## 修行法门

### 1. 持戒
- 五戒
- 十戒
- 菩萨戒
- 修仙戒律

### 2. 布施
- 财布施
- 法布施
- 无畏布施

### 3. 持咒
- 持诵真言
- 消业增福

### 4. 禅修
- 观想因果
- 内观自省

## 因果修炼的境界

### 1. 凡人境界
- 修五戒十善
- 积累小功德

### 2. 修士境界
- 修持戒律
- 积累中功德

### 3. 大乘境界
- 修菩萨行
- 积累大功德

### 4. 圆满境界
- 功德圆满
- 飞升仙界

## 因果与修行的关系

- 功德是**外功**
- 智慧是**内功**
- 内外兼修方为正道
