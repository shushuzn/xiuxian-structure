# 因果律

## 定义

**因果律**是天道运行的最高法则之一，规定**一切行为必有其后果**。因果律是仙侠世界最根本的运行法则。

## 核心法则

### 1. 种因得果
- 善因必有善果
- 恶因必有恶果
- 不因身份而改变

### 2. 因果不虚
- 任何行为都不消失
- 无论多小的行为

### 3. 果报不爽
- 善恶到头终有报
- 只争来早与来迟

### 4. 因果可转
- 通过忏悔可减轻
- 通过行善可抵消

## 因果律的表现

### 1. 天劫
- 恶业深重者天劫加重
- 善业圆满者天劫减轻

### 2. 心魔
- 恶念积累成心魔
- 心魔是业力的内化

### 3. 轮回
- 善业转生善道
- 恶业转生恶道

### 4. 奇遇
- 功德深厚者遇奇遇
- 恶业深重者遇灾祸

## 因果律的层次

| 层次 | 范围 | 时间 |
|---|---|---|
| 现世果 | 今生 | 立即 |
| 来世果 | 转世后 | 死后 |
| 累世果 | 多世累加 | 数百世 |
| 永恒果 | 不可消 | 永远 |

## 因果与命运

- 命运是因果的体现
- 修士通过修行可改命
- 大功德者可**逆天改命**

## 因果与法则

- 法则高于因果
- 因果是法则之一
- 修仙者最终需**跳出因果**
