# 化神期

## 定位
真正的高阶修士：神魂化形，可瞬移挪移。
化神期修士在凡界已属凤毛麟角，常闭关百年不出，是顶级宗门的底蕴所在。

## 等级划分
- **化神初期** — 神魂初化，可短暂瞬移
- **化神中期** — 神魂稳固，瞬移距离与精度大幅提升
- **化神后期** — 神魂大成，可御使灵宝
- **化神巅峰** — 触摸炼虚门槛，需悟透至少一种天地法则

## 寿命上限
约 2000 年（神魂不灭则生机长存）

## 核心能力
- **神识**：覆盖数十里，可离体远程攻击
- **法力**：可驱动灵宝，灵宝器灵可自主行动
- **飞行**：瞬移 / 空间挪移（极短距离）
- **神魂**：神魂与肉身相辅相成，可分神应对
- **丹药**：可承受仙品丹药
- **法则运用**：开始掌握天地法则的具体运用

## 突破条件（化神→炼虚）
- 神魂圆满，至少悟透一种天地法则
- 通常无外丹可助，全凭自身感悟
- 需在超级灵脉或特殊秘境突破
- 突破失败：神魂俱灭，肉身亦难存

## 限制
- 每次瞬移消耗极大，不可滥用
- 法则运用仍属"入门"，无法创造法则
- 肉身仍是软肋，受重创仍会陨落
