# 大乘期

## 定位
凡界修行的终点，飞升前的最后境界。
大乘期修士已跳出凡界常理，下一步要么飞升仙界，要么滞留凡界为一方巨擘。

## 等级划分
- **大乘初期** — 飞升劫后余生
- **大乘中期** — 法则大成，神魂与天地同寿
- **大乘后期** — 飞升准备完毕，可随时破界飞升
- **大乘巅峰** — 滞留凡界的极限，常被称为"半仙"

## 寿命上限
近乎无限（与天地同寿）

## 核心能力
- **神识**：覆盖十万里以上，可感知天地运转
- **法力**：法力几近无穷
- **飞行**：可随意穿梭各界
- **法则运用**：可创造大型法则领域，可短暂改写天地规则
- **不死不灭**：肉身毁可重塑，神魂灭可转世（限一次）

## 突破条件（大乘→飞升）
- 心境圆满、修为无瑕
- 破开界面壁障飞升仙界
- 飞升过程本身也是一种渡劫

## 限制
- 凡界停留越久，天地压制越强
- 不可干预凡界运转（否则遭天谴）
- 飞升仙界后需重新适应仙界法则
