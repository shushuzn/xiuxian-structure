# 天材地宝总览

## 定义

**天材地宝**是修仙界一切稀有资源的统称——灵草、灵木、灵金、灵水灵火、灵兽灵虫等。是**炼丹、炼器、布阵、绘符**的原材料。

## 分类总览

按形态分为 7 大类：

- **灵草类**：草本灵药，炼丹主料
- **灵木类**：木质灵材，多用于法器
- **灵金属类**：金属矿石，炼器核心
- **灵水灵火类**：液态/火焰型稀有资源
- **灵兽灵虫类**：活物资源（内丹、妖核、灵虫）
- **灵丹灵药原料**：炼丹的中间品
- **天外奇珍**：跨界稀有之物

## 等级（按年份/品质）

- **百年**：筑基期可用
- **千年**：结丹-元婴期
- **万年**：化神-合体期
- **十万年**：渡劫-大乘期
- **仙材**：飞升后仙界
- **神材**：神界专属

## 核心价值

1. **炼丹原料**：九转还魂丹、万年灵芝丹等
2. **炼器原料**：五行环、紫薇剑等
3. **阵法材料**：高阶阵旗、阵眼
4. **绘符材料**：符墨、符纸
5. **直接服用**：部分可临时提升修为
