# 灵丹灵药原料

## 定义

**灵丹灵药原料**是已经过初步处理的天材地宝，可直接作为炼丹辅料的中间品。

## 与天材地宝的区别

- 天材地宝：**原始形态**（灵草、矿石等）
- 灵丹原料：**已加工**（如：灵草粉末、矿石精华）

## 子类

- **灵草粉末**：灵草研磨后的粉末
- **金属精华**：矿石提纯后的精华液
- **妖兽血精**：妖血浓缩物
- **灵木树脂**：灵木流出的汁液
- **灵泉水**：灵泉提取的浓缩液

## 著名原料

### 灵草粉末
- **来源**：灵草
- **用途**：炼丹主料
- **保存**：需玉瓶封存

### 玄铁精华
- **来源**：玄铁
- **用途**：炼器主料
- **保存**：需灵火封存

### 妖王血精
- **来源**：妖王精血
- **用途**：血脉丹药
- **保存**：玉瓶+符箓

## 加工流程

1. 采集天材地宝
2. 初步提纯（去杂质）
3. 浓缩精华
4. 封存待用
