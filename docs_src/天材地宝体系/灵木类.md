# 灵木类

## 定义

**灵木**是蕴含灵性的木本植物，常用于法器炼制、阵法旗幡、特殊用途。

## 子类

- **常青灵木**：万年常青，炼器主料
- **雷击灵木**：经雷劫淬炼的灵木
- **魂木**：蕴含魂魄之力的灵木
- **化形灵木**：即将化形的灵木（极稀有）

## 著名灵木

### 万年灵木
- **等级**：万年
- **属性**：木
- **用途**：炼制高阶法器主料

### 雷击木
- **等级**：千年
- **属性**：雷
- **用途**：炼制雷属性法器、特殊符箓

### 养魂木
- **等级**：万年
- **属性**：魂
- **用途**：续命、养魂（南陇侯救妻用）

### 化形灵木
- **等级**：十万年
- **属性**：木
- **用途**：传说级法器核心

## 用途

1. **法器杆/柄**：剑柄、幡杆
2. **阵法旗幡**：阵旗主料
3. **炼丹辅料**：木属性丹药的药引
4. **制符材料**：符纸、符笔
