# 灵水灵火类

## 定义

**灵水**与**灵火**是特殊的液态/火焰型灵材，常用于炼丹、炼器、特殊功法辅助。

## 灵水子类

- **灵泉**：天然灵气凝聚的泉水
- **玄冰真水**：极寒之水
- **天一真水**：传说级灵水
- **黄泉水**：冥界之水
- **三昧真水**：仙家灵水

## 灵火子类

- **地火**：地底灵火
- **天火**：天降灵火
- **三昧真火**：仙家灵火
- **幽冥鬼火**：冥界之火
- **涅槃之火**：凤凰专属

## 著名灵水灵火

### 玄冰真水
- **等级**：上品
- **属性**：冰
- **用途**：冰属性炼丹、淬体
- **产地**：北极冰川

### 三昧真火
- **等级**：仙品
- **属性**：火
- **用途**：炼制仙器、炼丹
- **产地**：仙界

### 黄泉水
- **等级**：上品
- **属性**：冥
- **用途**：冥界炼丹、炼器
- **产地**：冥界忘川

### 涅槃之火
- **等级**：神品
- **属性**：火
- **用途**：凤凰涅槃
- **产地**：凤凰岭

## 用途

1. **炼丹**：火助药性，水调阴阳
2. **炼器**：淬炼法器
3. **淬体**：以水火淬炼肉身
4. **战斗**：直接作为攻击手段
