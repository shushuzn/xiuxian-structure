# 灵草类

## 定义

**灵草**是草本灵药的总称，是炼丹最常用的原料。形态为草本植物，富含灵性。

## 子类

- **根茎类**：人参、何首乌等
- **叶花类**：灵芝、九幽还魂草
- **果实类**：灵果、灵桃
- **菌类**：灵芝、太岁

## 著名灵草

### 九幽还魂草
- **等级**：千年
- **属性**：幽冥
- **用途**：炼制九转还魂丹
- **产地**：灵药岛

### 千年血参
- **等级**：千年
- **属性**：血
- **用途**：结丹期突破辅助
- **产地**：乱星海域

### 千年灵芝
- **等级**：千年
- **属性**：木
- **用途**：元婴期修炼辅助

### 万年太岁
- **等级**：万年
- **属性**：木
- **用途**：化神期延寿

## 采集要点

- 灵草**不可过度采集**，否则会断根
- 采摘要按**时令**（春生、夏长、秋收、冬藏）
- 部分灵草有**守护妖兽**
