# 灵金属类

## 定义

**灵金属**是蕴含灵性的金属矿石，是炼制法器、灵宝的核心材料。按品质分为凡铁、玄铁、寒铁、星辰铁等。

## 子类

- **凡铁**：普通灵铁，入门级法器
- **玄铁**：筑基级法器主料
- **寒铁**：冰属性法器
- **雷铁**：雷属性法器
- **星辰铁**：跨界天外之铁
- **仙铁**：仙器级
- **神铁**：神器级（极罕见）

## 著名灵金属

### 玄铁
- **等级**：中品
- **属性**：金
- **用途**：结丹级法器主料
- **产地**：玄铁矿脉

### 万年寒铁
- **等级**：上品
- **属性**：冰
- **用途**：冰属性法宝
- **产地**：北极之地

### 星辰铁
- **等级**：仙品
- **属性**：空间
- **用途**：仙器核心
- **产地**：天外秘境

### 九天玄铁
- **等级**：上品
- **属性**：金
- **用途**：剑修至宝

### 神铁
- **等级**：神品
- **属性**：混沌
- **用途**：传说级神器

## 冶炼要点

- 需以**真火**冶炼
- 锻造时需配合**符箓**加持
- 部分需**雷劫淬炼**
