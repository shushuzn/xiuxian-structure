# 魂契

## 定义

以双方**灵魂碎片**直接融合签订的极端契约——绑定最深、约束最强、代价最大。
魂契可适用任何对象（人、灵兽、法器、势力），但因代价过大被视为禁忌，仅在最极端情况下使用。
魂契一旦签订，双方**几乎无法分割**——一方死亡，另一方必然重创或同死。

## 机制

### 签订方式

- **灵魂融合** — 双方各取一丝灵魂碎片，融合后烙印在彼此神魂中
- **代价**：
  - 双方修为各倒退 1-3 个小境界（灵魂损失）
  - 双方共享部分寿元（折寿）
  - 双方日后突破瓶颈概率降低（灵魂不再完整）
- **仪式**：极危险，需双方灵魂在仪式中短暂"出窍"并融合
- **耗时**：数刻钟（失败则双方灵魂受创）

### 契约效果

魂契是契约体系的**极致形态**——除具备灵魂契约的所有效果外，还有：

- **灵魂共享** — 双方法力可互通（一方耗尽时可借另一方）
- **感知即时** — 双方感知对方的精度达到"心灵感应"级别
- **生死同步** — 一方死亡，另一方必然重创或同死
- **共同命运** — 一方渡劫，另一方会被动承受部分天劫压力

## 适用对象

- **道侣 / 至亲** — 极少见，多用于生死与共的伴侣
- **本命至交** — 极少数修士间的"同生共死"
- **极端仪式** — 邪修常用（牺牲他人签订魂契）

## 优劣

### 优势

- **最牢固** — 任何情况下都不可背弃
- **最深协同** — 战斗配合度超越灵魂契约
- **资源共享** — 法力互通、寿元共享

### 劣势（极端）

- **代价极大** — 修为倒退、寿元缩短、突破概率降低
- **同死风险** — 一方死亡，另一方重创或同死
- **不可解除** — 几乎不可解除（需双方同死）
- **禁忌** — 正道视为禁忌，仅在最极端情况下使用

## 禁忌用法

### 强行魂契（邪修）

- **机制**：强行从对方灵魂中抽取碎片签订魂契
- **后果**：被签者必死（灵魂被撕裂）
- **稀有度**：极罕见，被修仙界共同唾弃

### 魂契转嫁

- **机制**：将已有的魂契转嫁到第三方
- **后果**：原双方均重创，第三方可能死亡
- **稀有度**：传说级邪法

### 魂契吞噬

- **机制**：利用魂契共享灵魂的特性，吞噬对方灵魂
- **后果**：被吞噬者必死，施术者修为暴涨但灵魂不纯
- **稀有度**：极罕见邪法

## 关键剧情节点

- **历史案例**：修仙史上仅记载过 3 次成功签订的魂契（道侣、至交、母子）
- **禁忌警示**：90% 的魂契都以双方同死告终

## 解除契约

- **几乎不可解除** — 需双方同时死亡
- **传说级秘法** — 极少数化神期以上修士掌握
- **代价**：即便解除，双方均修为大损
