# 妖丹（妖兽核心资源）

## 定位
妖兽体内凝聚的精华，是妖修的核心资源。
可作为法器材料、灵药炼丹、阵法能量。

## 等级

| 等级 | 对应境界 | 稀有度 |
|---|---|---|
| 一~三级 | 练气级 | 常见 |
| 四~六级 | 筑基级 | 较常见 |
| **七~九级** | **结丹级** | **罕见** |
| **化形期** | **元婴级** | **极罕见** |
| 顶级 | 化神级 | 传说 |

## 核心用途

### 炼丹
- **化形丹**：需化形期妖丹作为主药
- **妖修突破辅助丹**：需对应等级妖丹

### 阵法
- **妖丹激活**：部分阵法需妖丹激活
  - 雷师之墓入口需 3 颗七级以上妖丹
  - 化形期妖丹可作为灵药岛入岛信物

### 法器
- **妖丹嵌入**：部分高阶法器需嵌入妖丹
- **妖丹炼化**：可将妖丹炼化为一次性攻击

### 交易
- **潮汐宫官方收购价**：化形期妖丹 100 枚中品灵石
- **黑市**：价格翻倍

## 来源
- **猎杀妖兽**：从妖兽体内取出
- **自然死亡**：妖兽死后妖丹仍在
- **化形失败**：化形失败时妖丹可能破碎

## 提取方法
- 妖兽死后 7 日内提取（逾期妖丹会腐朽）
- 需以特殊手法破开妖兽丹田
- 筑基期以上修士可操作

## 关键剧情节点
- **vol2**：韩立猎杀墨渊蛟获取七级妖丹作为雷师之墓入口钥匙
- **vol2**：韩立需化形期妖丹续养魂木
- **aoyue**：妖修结盟中化形期妖丹是核心资源
