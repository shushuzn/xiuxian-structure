# 妖兽

## 定义
天地灵气滋养的兽类，开启灵智、吸收日月精华后可修炼成妖。
妖兽是修仙界重要的资源（内丹、皮毛、兽血）与威胁（袭击修士、争夺灵脉）。

## 构成要素

### 品级（对应修士境界）

| 品级 | 对应修士境界 | 战力 |
|---|---|---|
| 凡兽 | 凡人 | 与凡人相当 |
| 灵兽 | 炼气期 | 普通妖物 |
| 妖兽 | 筑基期 | 中阶威胁 |
| 大妖 | 结丹期 | 区域性霸主 |
| 妖王 | 元婴期 | 一方巨擘 |
| 妖皇 | 化神期 | 顶级存在 |
| 妖帝 | 炼虚期及以上 | 传说级 |

### 属性
- **五行属性兽**：对应金木水火土灵根修士
- **变异属性兽**：雷、冰、风等变异属性，极为稀有
- **无属性兽**：可适应多种灵气环境

### 内丹
- 妖兽修为的核心，类比人类金丹/元婴
- 内丹可入药、炼器，是高阶炼丹炼器的珍贵原料
- 高阶妖兽的内丹价值连城

### 化形
- 修为达到"妖王"（元婴期）后可化为人形
- 化形不完全时保留兽类特征（耳朵、尾巴等）
- 化形后可混入人类社会，但也受天劫制约

### 灵智
- **懵懂**：本能行事
- **初开**：能听懂人言
- **通灵**：可与修士交流
- **化形**：拥有人类智慧与思维
