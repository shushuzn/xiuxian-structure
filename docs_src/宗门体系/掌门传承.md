# 掌门传承

## 定义

**掌门传承**是宗门最高权力交接的过程，是宗门最重要的事件之一。

## 传承方式

### 1. 父子传承
- **适用**：家族式宗门
- **优点**：稳定
- **缺点**：可能选错人

### 2. 师徒传承
- **适用**：师徒型宗门
- **优点**：培养成熟
- **缺点**：可能绝后

### 3. 长老会推举
- **适用**：大型宗门
- **优点**：民主
- **缺点**：可能分裂

### 4. 实力为王
- **适用**：魔道宗门
- **优点**：强者为尊
- **缺点**：血腥

### 5. 飞升禅让
- **适用**：飞升型宗门
- **场景**：掌门飞升前指定继承人
- **优点**：稳定过渡

## 传承流程

1. **宣布继承**：现任掌门公开指定
2. **长老会认可**：长老会多数通过
3. **公开继位**：举行继位大典
4. **传功传位**：传授掌门信物

## 掌门信物

- **掌门令牌**：身份证明
- **宗门功法传承**：核心功法
- **护山阵法钥匙**：操控护山大阵
- **秘传灵宝**：历代掌门灵宝

## 著名传承案例

### 落云宗
- 上代掌门：云姓化神期（飞升）
- 现任掌门：云姓化神期（指定继任者）

### 太一宫
- 上代掌门：太一真人
- 现任掌门：太一真人（飞升前夕指定）

## 冲突与处理

- **争议继承**：由长老会仲裁
- **篡位**：被视为大罪
- **多继承人**：由长老会决定
