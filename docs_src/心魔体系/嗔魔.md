# 嗔魔

## 定位

心魔四类中最凶险的一种：源于修士的**怨愤、嗔怒、杀心**。
嗔魔一旦触发，修士会陷入暴怒与嗜血的失控状态，敌我不分，甚至自毁道基。
与贪魔的"幻境沉溺"不同，嗔魔是"实境暴走"——对修士肉身和神魂都有即时伤害。

## 等级与触发

- **微嗔** — 对仇敌的愤恨，炼气→筑基期常见
- **大嗔** — 对不公世道的怨愤，筑基→结丹期高发
- **极嗔** — 对天地法则的不满，结丹→元婴期高发（"天道不公，何必修仙"）
- **至嗔** — 恨天恨地恨己，化神→炼虚时若被勾起，可直接走火入魔

## 表现

- 突破时双目赤血，神识狂暴
- 体内法力逆乱，金丹震颤欲裂
- 产生强烈的破坏欲——即便身旁是同门也欲杀之
- 严重时口喷鲜血、肉身龟裂
- 渡劫中触发嗔魔 = 主动招雷，几乎必死

## 高发人群

- 杀伐重的散修（手上沾血过多）
- 经历过灭门、灭族之仇的修士
- 修炼攻击性功法的剑修、魔修
- 经历过重大背叛、心怀执念者

## 应对

- **清心丹** — 仅对中低阶嗔魔有效
- **静心咒 / 清心诀** — 佛门、道门专用
- **远离仇怨** — 突破前需闭关清修，断绝与仇家的接触
- **师长护法** — 嗔魔暴走时需旁人强力镇压

## 关键剧情节点

- **aoyue**：aoyue 渡劫时正遇墨大夫来袭，嗔魔与外敌夹击，神魂俱碎
- **韩立**：韩立多次强压对墨大夫的杀心，反而被嗔魔险些反噬
