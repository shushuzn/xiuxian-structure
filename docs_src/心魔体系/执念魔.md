# 执念魔

## 定位

心魔四类中最难破的一种：源于修士对**某人、某事、某物**的执念，无法放下。
执念魔与贪魔的区别在于：贪是对"未得"的渴望，执念是对"已失"或"将失"的恐惧与不甘。
执念魔往往伪装最深——修士自己都不愿承认这是执念。

## 等级与触发

- **微执** — 执念于某件物品（如父母遗物、道侣遗物）
- **大执** — 执念于某个人（道侣、亲人、仇人）
- **极执** — 执念于某段往事（灭门、失恋、师徒反目）
- **至执** — 执念于"我一定要做到"（发誓复仇、誓证长生）

## 表现

- 突破时神识陷入过去的画面，反复重演
- 修士明知是幻境，但不愿醒来
- 体内法力停滞，魂魄不肯离体或成形
- 即便渡过雷劫，因心境有漏，日后必有劫数

## 高发人群

- 经历过重大情感变故的修士
- 背负师门、家族遗命的修士
- 立下大誓愿者（"不成道，不出山"）

## 应对

- **清心丹 + 凝神丹** — 联合使用
- **完成执念** — 在闭关前主动了结（如报仇、还愿、放下）
- **转化执念** — 将执念转化为"愿力"（如"为师门复仇"转为"为师门开枝散叶"）
- **红尘炼心** — 主动入世，体验执念之虚妄

## 关键剧情节点

- **韩立**：韩立对南宫婉的执念在结丹期多次险些让其走火入魔
- **aoyue**：aoyue 对韩立的执念（前世情缘）让其渡劫时心境崩溃
