# 法相天地

## 概述

法相天地是修士以神魂能量凝聚的巨大虚影，是肉身与法则的具象化表达。法相的形态与修者的道法领悟、神魂强度密切相关——剑修的法相通常是巨剑，体修的法相往往是远古巨兽，佛修的法相则是金刚佛陀。法相一旦展开，修士战力暴涨，但消耗同样巨大。法相品质是衡量修者底蕴的重要标志。

## 等级/分类

法相按境界要求与属性分为以下类型：

- **金刚法相** — 结丹期可修，防御型，召金刚虚影护体，大幅提升双抗
- **灭世法相** — 元婴期可修，攻击型，化身修罗虚影，攻击暴增但消耗寿元
- **青莲法相** — 化神期可修，净化型，驱散诅咒与心魔，辅助为主
- **无相法相** — 炼虚期可修，变化型，可模仿对手招式，以彼之道还施彼身
- **鲲鹏法相** — 合体期可修，速度型，化鲲鹏之翼，瞬息千里
- **真龙法相** — 渡劫期可修，威压型，真龙降世震慑低阶
- **仙麟法相** — 大乘期可修，祥瑞型，增幅气运，附带因果攻击
- **神凰法相** — 仙人境可修，涅槃型，濒死触发涅槃重生

## 效果/特性

法相的核心机制：

- **法相与神魂绑定** — 法相受伤会反馈到修者神魂，重创可导致修为跌落
- **法相大小与境界正相关** — 结丹期法相不过数丈，仙人境可达千丈
- **法相可融合器灵** — 拥有器灵的法宝可融入法相，提升法相灵性与自主性
- **法相对抗规则** — 双方法相对撞时，法相品质与神魂强度决定胜负
- **法相持续时间** — 受法力储备与神魂耐力限制，通常不超过一柱香
- **法相进阶** — 法相可随修为提升而进化，形态与能力均可蜕变

## 实战运用

法相在战斗中的典型运用场景：

- **开场压制** — 战斗初期展开法相可形成威慑，压制对手战意
- **防御反击** — 以防御型法相抵挡对手爆发，待对手法力消耗后反击
- **法相化器** — 高阶修士可将法相凝聚为实体兵器，威力堪比灵宝
- **法相自爆** — 危急时刻引爆法相可造成大范围毁灭性伤害，但修者神魂重创

## 法相修炼

法相修炼的三个关键阶段：

- **凝形** — 以神魂凝聚法相基础形态，此阶段法相脆弱易碎
- **固化** — 法相可稳定维持，具备实战能力，可与对手正面对抗
- **化道** — 法相融入自身道法，可随心意自由变化形态
