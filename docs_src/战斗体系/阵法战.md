# 阵法战

## 概述

阵法战是将阵法运用于实战的战术体系。与常规阵法不同，战斗阵法强调快速布设、灵活变阵与实时操控。阵法战可一人成阵（随身阵盘），也可多人合阵（战阵）。高阶修士甚至可做到在战斗中临时布阵，以阵困敌、以阵杀敌。阵法战是修仙界中最讲究策略与战术配合的战斗方式。

## 等级/分类

常见的战斗阵法按用途分类：

- **九霄雷阵** — 雷系攻击阵，引天雷持续轰击，对魔道修士杀伤加倍，被避雷法宝克制
- **天机算阵** — 推算型，预判对手所有动作，被因果干扰或混沌遮蔽克制
- **八卦迷踪** — 迷幻型，困敌于阵内无法脱身，被破阵神识或天眼通克制
- **五行生灭** — 平衡型，五行相生循环，攻防一体，五行俱全者可平分秋色
- **万剑归宗** — 攻击型，万剑齐发连绵不绝，重甲防御或剑域对冲可破
- **周天星斗** — 星辰型，威力与星象位置相关，昼时被压制，遮蔽天机可破
- **太虚幻境** — 幻阵，困敌于执念幻象，清心诀或佛门慧眼可解
- **混沌四象** — 上古阵，引混沌之力化四象神兽，需规则级破法才能破解

## 效果/特性

阵法战的特色机制：

- **变阵** — 战斗阵法可根据战局切换阵型，灵活应对不同对手
- **阵眼强化** — 阵眼处施法者战力翻倍，但阵眼暴露后易被针对性破坏
- **合击增幅** — 多人协作阵法可叠加战力，每增加一人，阵法威力非线性增长
- **反制与克制** — 每个阵法都有对应的克制手段，知己知彼可破阵
- **阵盘便携** — 高阶阵法可预制为阵盘，战斗中瞬间展开，大幅提升战术灵活性
- **阵法师为核心** — 阵法战极度依赖阵法师的临场判断与操控能力

## 战阵配合

多人阵法战的基本配合方式：

- **防御阵型** — 前排主防，后排输出，阵法师居中调控
- **攻击阵型** — 全体以攻击为主，阵法师以增幅阵强化输出
- **困敌阵型** — 利用迷阵困住敌人，分割战场，逐个击破
- **混战阵型** — 各自为战但保持阵法连接，阵法能量可自由调配

## 阵法师在战斗中的角色

阵法师在团队战斗中的核心职责：

- **布阵指挥** — 负责快速布置战斗阵法，指示队员站位配合
- **阵眼守护** — 阵法师通常是阵眼，需要队友保护不被对手针对性攻击
- **变阵时机** — 根据战局变化及时切换阵法类型，打乱对手节奏
- **资源管理** — 合理分配灵石与阵盘消耗，确保阵法持续运转
