# 领域

## 概述

领域是修士以自身道韵与法则领悟影响周围空间形成的特殊场域。领域范围内，施法者可以获得全方位的战斗优势——属性加成、敌人削弱、规则改变。领域的形成需要修者对某种法则有足够深的领悟，通常是化神期及以上修士的标志性能力。领域是衡量修士法则感悟深度的重要标准。

## 等级/分类

领域按品级分为八类，领域范围与效果随品级提升：

- **火域** — 地级，百丈范围，烈焰焚天，火属性功法倍增
- **冰域** — 玄级，五十丈范围，冻结法力流动，减速削弱敌人
- **雷霆域** — 地级，八十丈范围，雷霆密布，破邪克魔附带麻痹
- **剑域** — 天级，百丈范围，剑意纵横，可操控万千飞剑
- **重力域** — 玄级，三十丈范围，重力倍增，压制移动与飞行
- **时间域** — 仙级，十丈范围，改变时间流速，逆天但范围极小
- **灭界域** — 神级，千丈范围，规则归零，仅凭肉身与意志战斗
- **生灵域** — 圣级，范围随心，领域即小世界，操控一切规则

## 效果/特性

领域的实战应用：

- **领域压制** — 领域内敌人全属性下降，下降幅度与双方境界差正相关
- **领域对抗** — 双领域重叠时发生碰撞，品级高者覆盖品级低者
- **领域消耗** — 维持领域持续消耗法力，领域越大消耗越快
- **领域融合** — 可将领域与阵法结合，形成永久性阵域
- **神识决定领域精度** — 神识越强，领域内操控越精细
- **领域扩张** — 领域范围可通过修炼扩大，极限范围受境界与法则领悟双重制约

## 领域修炼

领域的修炼路径通常为：

- **雏形期** — 初步感知法则，领域范围极小且不稳定
- **成形期** — 领域可稳定维持，具备基础压制效果
- **精通期** — 领域内可进行精细操控，如选择性豁免友方
- **领域化界** — 领域可短暂独立于外界法则，形成伪独立空间
- **领域成道** — 领域演化为小世界，可容纳生灵，是通往神帝级的必经之路
