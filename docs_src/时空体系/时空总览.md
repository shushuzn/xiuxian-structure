# 时空总览

## 定义

**时空**是宇宙最根本的两大法则——**时间**与**空间**。是修仙界最神秘、最强大的法则之一，高阶修士方能触及。

## 核心法则

### 时间法则
- **定义**：万物运转的节奏
- **体现**：生老病死、季节轮回
- **掌握者**：时间系大能

### 空间法则
- **定义**：万物存在的维度
- **体现**：距离、范围、维度
- **掌握者**：空间系大能

### 时空统一
- **关系**：时间与空间互为表里
- **体现**：空间扭曲影响时间流速
- **终极**：时空法则的完全掌握

## 时空的重要性

### 1. 修炼
- 时间法则加速/减速修炼
- 空间法则瞬移、开辟洞天

### 2. 战斗
- 时间类攻击减速敌人
- 空间类攻击困锁敌人
- 时空类为**至高战斗**

### 3. 飞升
- 时空法则助飞升
- 可在时间长河中寻找飞升契机

## 时空修炼的难度

| 境界 | 时空能力 |
|---|---|
| 化神期 | 触及皮毛 |
| 炼虚期 | 初步感知 |
| 合体期 | 可运用 |
| 渡劫期 | 较熟练 |
| 大乘期 | 掌握 |
