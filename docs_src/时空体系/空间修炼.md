# 空间修炼

## 定义

**空间修炼**是通过领悟**空间法则**提升修为的法门。是中阶修士的进阶之路。

## 修炼境界

### 1. 空间感知
- **境界**：元婴期
- **能力**：感知空间结构
- **修炼**：观察空间裂缝

### 2. 储物法器
- **境界**：结丹期
- **能力**：炼制储物袋
- **修炼**：炼器

### 3. 空间瞬移
- **境界**：化神期
- **能力**：短距离瞬移
- **修炼**：空间法则入门

### 4. 空间洞天
- **境界**：炼虚期
- **能力**：开辟个人洞天
- **修炼**：空间法则掌握

### 5. 空间禁锢
- **境界**：合体期
- **能力**：空间禁锢敌人
- **修炼**：空间法则精通

### 6. 空间湮灭
- **境界**：渡劫期
- **能力**：空间湮灭
- **修炼**：空间法则圆满

## 修炼法门

### 1. 炼器法
- 炼制储物法器
- 炼制空间法器
- 炼制空间阵法

### 2. 开辟法
- 开辟洞天
- 开辟秘境
- 开辟个人世界

### 3. 移动法
- 短距离瞬移
- 跨界穿行
- 跨界面传送

## 空间法器

- **储物袋**：入门级
- **储物戒**：常用
- **洞天珠**：内含小世界
- **空间之镜**：跨界面传送

## 风险

- 空间裂缝可能**吞噬**
- 空间折叠可能**迷失**
- 跨界穿行可能**卡在壁障中**
