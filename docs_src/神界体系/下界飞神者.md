# 下界飞神者

## 定义

从下界（人界或仙界）渡过神劫进入神界的存在——神界最特殊的群体。
下界飞神者自带下界因果与修炼经验，与神界本土神人（神兽化形等）有显著区别。
下界飞神者多从「凡人神格」起步，需在神界从零开始。

## 飞神路径

### 直接飞神（罕见）

- **路径**：人界 → 飞升 → 仙界 → 神界
- **难度**：极高，需数百万年积累
- **代表**：传说级存在

### 间接飞神（典型）

- **路径**：人界 → 飞升 → 仙界修炼 → 渡神劫 → 神界
- **时间**：飞升后数百万年
- **代表**：大飞升者

## 飞神者分类

| 类型 | 概率 | 后续 |
|---|---|---|
| 普通飞神者 | 80% | 凡人神格，需重头开始 |
| 天才飞神者 | 15% | 仙级神格，可直接任神官 |
| 传说飞神者 | 4% | 真神格，可开宗立派 |
| 神帝飞神者 | 1% | 大神格，需谨慎培养 |

## 飞神者特点

### 优势

- **下界经验** — 经历过多次天劫，意志坚定
- **多体系修炼** — 接触过下界多种体系
- **因果清晰** — 因果分明，不会胡搅蛮缠

### 劣势

- **神气不适** — 初到神界需数十年「神气适配」
- **下界因果** — 与下界亲人的因果羁绊
- **本土歧视** — 部分神界本土神人歧视下界飞神者
- **资源匮乏** — 初到神界多「身无分文」

## 飞神者与神界关系

### 适应期

- 飞神后需数十年「神气适配」
- 适配期修炼效率较低
- 适配后修炼速度提升百倍

### 长期发展

- 修为提升后可获神职
- 部分下界飞神者成为神界大佬
- 飞神者的「下界经验」是独特优势

## 历史案例

- **传说飞神者** — 雷师（8000 年前从下界飞升仙界后渡神劫）
- **神帝飞神者** — 极少数，多为下界天才
