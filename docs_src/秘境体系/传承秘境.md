# 传承秘境

## 定义

**传承秘境**是上古大能刻意留下，**专门用于传承衣钵**的秘境。通常设有考验关卡，**通过者得传承**，**失败者受惩处**。

## 核心特征

1. **有明确传承目标**：功法、法器、心得
2. **设有考验关卡**：智力、实力、心性多维考验
3. **传承对象有筛选**：可能限特定灵根、性别、时代

## 类型

### 功法传承
- 传承内容：功法秘籍
- 考验：修炼潜力、悟性
- 实例：某大能的洞府

### 法器传承
- 传承内容：本命法器
- 考验：法器契合度
- 实例：雷师之墓

### 心法传承
- 传承内容：道心感悟
- 考验：心性、悟性
- 实例：道德经残页所在

## 著名实例

### 雷师之墓
- **等级**：宝品（结丹级）
- **考验**：雷法考验
- **传承**：雷师印 + 雷法
- 见 [雷师之墓](雷师之墓.md)

### 散修遗迹
- **等级**：灵品（筑基级）
- **考验**：综合实力
- **传承**：散修心得

## 进入与失败代价

- 多数传承秘境**永久开放**
- 失败可能被困、被封、甚至形神俱灭
- 部分传承设有**一次性激活**，激活后永不再开
