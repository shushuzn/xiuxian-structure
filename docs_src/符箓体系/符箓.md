# 符箓

## 定义
绘制于符纸上的法术载体，激发后可在短时间内释放预先封印的法术。
符箓是炼气期和筑基期修士最常用的辅助手段，弥补自身法力不足。

## 构成要素

### 品级

| 品级 | 蕴含法术威力 | 绘制者修为 |
|---|---|---|
| 凡符 | 炼气级法术 | 炼气期 |
| 灵符 | 筑基级法术 | 筑基期 |
| 宝符 | 结丹级法术 | 结丹期 |
| 仙符 | 元婴级及以上法术 | 元婴期+ |

### 类别

| 类别 | 用途 |
|---|---|
| **攻击符** | 激发攻击法术，如火球符、雷符 |
| **防御符** | 激发护罩，如金刚符 |
| **辅助符** | 加速、增幅、隐匿 |
| **遁符** | 瞬间移动、逃命 |
| **封印符** | 封印妖兽或物品 |
| **传讯符** | 远距离传信 |

### 构成要素
- **符纸**：载体，普通黄纸到灵纸
- **符墨**：绘制用的特殊墨水，常含灵药
- **符笔**：绘制工具，灵笔可提升成功率
- **符文**：核心，决定符箓功效

### 制作方式
- **手工绘制**：制符师亲手绘制，品质可控
- **批量印制**：用模板大批量制作，品质一般
- **一次成型**：以法力直接凝聚，省材料但难复刻

### 限制
- 符箓是一次性消耗品
- 长期存放会自然损耗
- 高阶符箓绘制失败率高
- 受绘制者水平限制，上限明显
