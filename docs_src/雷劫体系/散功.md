# 散功

## 定义

修士渡劫失败后，**修为尽失**或**形神俱灭**的总称。
是雷劫体系的"反面"——与[雷劫淬体](雷劫淬体.md)的升华相反，散功是彻底的毁灭或退化。
雷劫失败的代价因阶段不同而不同，越是高阶雷劫，散功越彻底。

## 构成要素

### 等级划分

#### 轻散功（部分修为丧失）

- 雷劫前期（前 1/3）失败
- 后果：当前境界修为尽失，退回前一境界
- 例：三九天劫第 1-3 道失败 → 结丹退回筑基，但金丹未碎，可重修
- 概率：约 20% 的渡劫失败属此类

#### 重散功（道基尽毁）

- 雷劫中期（中间 1/3）失败
- 后果：当前境界、道基、根基尽毁，沦为凡人或低阶修士
- 例：六九天劫第 7-12 道失败 → 元婴尽毁，神魂重创，沦为废人
- 概率：约 50% 的渡劫失败属此类

#### 形神俱灭（彻底消亡）

- 雷劫后期（最后 1/3）失败
- 后果：肉身、神魂、法力、元婴/元神一同毁灭，连转世机会都没有
- 例：九九天劫第 19-27 道失败 → 形神俱灭
- 概率：约 30% 的渡劫失败属此类，且无法逆转

### 触发条件

- 雷劫本身威力超过肉身、神魂承受极限
- **心魔叠加** — 突破时心魔爆发，与雷劫叠加致死
- **外敌干扰** — 渡劫时遭敌袭（如 aoyue 案例）
- **资源不足** — 灵石、丹药耗尽，无以为继
- **因果清算** — 杀业、因果太重，雷劫威力倍增

### 与心魔的关系

- 心魔常在雷劫中同时爆发
- 心魔先行 → 修士失守 → 雷劫趁虚而入 → 散功
- 雷劫先降 → 修士神魂受创 → 心魔趁机而起 → 散功
- 两者形成恶性循环

## 应对

### 渡劫前

- **心境磨练** — 减少心魔爆发的可能
- **资源储备** — 大量灵石、丹药
- **师长护法** — 防止外敌干扰
- **占卜吉凶** — 选择灵气浓郁、隐蔽的渡劫之地

### 渡劫中（散功已发生时）

- **轻散功可救** — 同阶修士强行介入，护下肉身，代价是护法者也受重创
- **重散功难救** — 需千年灵药 + 高阶丹药，救下后多为废人
- **形神俱灭不可救** — 即便师长在场也只能眼睁睁看着

## 关键剧情节点

- **aoyue**：aoyue 渡三九天劫第 7 道时，墨大夫来袭，嗔魔与外敌夹击，渡劫失败身死（形神俱灭）
- **历史案例**：修仙史上约 70% 试图突破元婴期的结丹修士以散功告终
