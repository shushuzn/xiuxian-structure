# 飞升

## 定义

大乘期修士渡过[飞升之劫](飞升之劫.md)，离开下界（人界），进入[仙界](仙界.md)的过程。
飞升是修仙之路的最高目标，也是修仙者从「凡人」向「仙人」的本质跃迁。
飞升**不可逆**——一旦飞升，几乎无法再回到下界（除非付出极大代价穿越[界面壁障](界面壁障.md)）。

## 构成要素

### 本质

- **大境界的终极考验** — 飞升之劫是修仙者最后一次天劫
- **本质跃迁** — 飞升后生命层次发生根本变化，寿元近乎无限
- **不可逆** — 飞升后几乎无法返回下界（参考 [界面壁障](界面壁障.md)）
- **代价巨大** — 飞升过程中会失去部分下界因果与羁绊

### 飞升三阶段

#### 1. 飞升之劫（最关键）

详见 [飞升之劫](飞升之劫.md)：
- 渡劫期→大乘期突破时承受
- 九九天劫的极巅（27 道雷的最后 3 道为飞升之劫）
- 失败：形神俱灭（无法转世）
- 成功：飞升仙界

#### 2. 飞升过程

飞升之劫渡过后的「过渡期」：
- 时长：数刻至数日不等
- 现象：
  - 周身仙光笼罩
  - 灵气（仙气）灌体
  - 肉身、神魂、法力被「仙化」
  - 与下界的因果逐渐剥离
- 风险：过程未完成时被强行打断会前功尽弃

#### 3. 仙界落地

- 落地位置：仙界接引台（所有飞升者的统一入口）
- 接引仙官：负责登记、分配住处、初步考核（参考 [仙职](仙职.md)）
- 初始身份：下界飞升者（参考 [下界飞升者](下界飞升者.md)）

### 飞升条件

#### 主观条件

- 修为达到渡劫期→大乘期突破临界
- 心境圆满（无执念魔隐患）
- 道心坚固（接受仙界新规则）

#### 客观条件

- 渡过[飞升之劫](飞升之劫.md)
- 有足够法力支撑飞升过程（数刻至数日）
- 不在下界有未了的「因果锁链」（如未报的杀父之仇）

### 飞升后果

#### 正面

- **寿元近乎无限**（参考仙界规则）
- **获得仙籍**（仙界正式居民）
- **可修炼仙级功法**（参考 [仙界功法](仙界功法.md)）
- **可使用仙灵石**（参考 [仙灵石](仙灵石.md)）

#### 负面

- **与下界因果断绝** — 无法直接干预下界事务
- **界面壁障** — 几乎无法返回下界（参考 [界面壁障](界面壁障.md)）
- **下界羁绊消散** — 与下界亲友的感应逐渐模糊
- **需重新适应** — 仙界规则与下界完全不同

### 飞升失败的常见原因

- **心魔未除** — 飞升过程中心魔爆发
- **法力不足** — 飞升过程中耗尽
- **外敌干扰** — 飞升时被偷袭
- **界面壁障** — 被界面壁障反噬
- **因果锁链** — 被未了因果强行拽回下界

### 飞升成功率

- 修仙史上：**约 10%** 渡劫期修士成功飞升
- 关键因素：心境 > 法力 > 资源
