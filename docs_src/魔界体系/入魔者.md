# 入魔者

## 定义

从人界 / 仙界 / 神界堕入魔界者——魔界最特殊的群体，与下界飞神者对应。
入魔者自带下界因果与修炼经验，与魔界本土魔族（魔兽化形等）有显著区别。
入魔者多从「凡人魔格」起步，需在魔界从零开始。

## 入魔路径

### 直接入魔（罕见）

- **路径**：人界 → 修仙 → 渡魔劫 → 魔界
- **难度**：极高，需斩断仙 / 神因果
- **代表**：传说级存在

### 间接入魔（典型）

- **路径**：人界 → 修仙 → 飞升 → 仙 / 神界 → 渡魔劫 → 魔界
- **时间**：飞升后数百万年
- **代表**：大入魔者

## 入魔者分类

| 类型 | 概率 | 后续 |
|---|---|---|
| 被逼堕魔 | 50% | 凡人魔格，需重头开始 |
| 主动入魔 | 30% | 仙级魔格，可直接任魔官 |
| 仙人叛入 | 15% | 真魔格，可开宗立派 |
| 魔帝转世 | 5% | 大魔格，需谨慎培养 |

## 入魔者特点

### 优势

- **下界经验** — 经历过多次天劫，意志坚定
- **多体系修炼** — 接触过下界多种体系
- **因果清晰** — 因果分明（但入魔后扭曲）

### 劣势

- **魔气不适** — 初到魔界需数十年「魔气适配」
- **下界因果** — 与下界亲人的因果羁绊（需斩断）
- **本土歧视** — 部分魔界本土魔族歧视入魔者
- **资源匮乏** — 初到魔界多「身无分文」

## 入魔者与魔界关系

### 适应期

- 入魔后需数十年「魔气适配」
- 适配期修炼效率较低
- 适配后修炼速度提升百倍

### 长期发展

- 修为提升后可获魔职
- 部分入魔者成为魔界大佬
- 入魔者的「下界经验」是独特优势

## 历史案例

- **被逼堕魔者** — 多数仙人 / 神人因仇家追杀被迫入魔
- **魔帝转世** — 极少数，多为下界天才转世

## 备注

- 入魔者总数远少于下界飞神者
- 入魔者多为悲剧人物
- 魔帝转世是最稀有入魔者
