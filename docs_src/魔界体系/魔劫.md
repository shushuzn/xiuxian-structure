# 魔劫

## 定义

仙人 / 神人入魔时承受的**魔界天劫**——比仙劫、神劫更扭曲、更凶险。
魔劫是入魔的最后一道关卡，渡过即入魔界，失败即魔化失败而亡（彻底消散）。
魔劫的次数与强度因人而异，但都遵循「魔道审判」的扭曲法则。

## 等级

- **品级**：至高阶·特殊
- **类型**：复合型（噬心 + 天魔 + 血魔 + 欲魔 + 心魔）
- **强度**：每道劫都相当于一次飞升之劫的扭曲版

## 魔劫类型（5 类）

### 噬心魔劫（最基础）

- **形态**：魔气噬心（噬咬神魂）
- **频率**：每入魔者必渡
- **渡过**：斩断仙 / 神因果，奠定魔修基础
- **风险**：噬心失败会丧失神智

### 天魔劫

- **形态**：天魔降世（具象化的魔界守护者）
- **频率**：5000 年一次
- **渡过**：魔躯转化为魔体
- **风险**：天魔会撕咬魔魂

### 血魔劫

- **形态**：血海涌现（需在血海中坚持七日）
- **频率**：10000 年一次
- **渡过**：魔魂淬炼为魔魂
- **风险**：血海会吞噬意志薄弱者

### 欲魔劫（最凶险）

- **形态**：欲魔具象化（可能是毕生执念的扭曲）
- **频率**：20000 年一次
- **渡过**：斩断最后执念
- **风险**：欲魔会让人沉沦于欲望

### 心魔劫（终极）

- **形态**：具象化的心魔（仙 / 神界心魔的扭曲版）
- **频率**：50000 年一次
- **渡过**：超脱仙 / 神道心，重塑魔道心
- **风险**：若无法承受，可能永远沉沦于心魔

## 渡过收益

- **魔体** — 仙体 / 神体转化为魔体
- **魔魂** — 仙魂 / 神魂转化为魔魂
- **魔力** — 法则扭曲融合，可施展魔功
- **魔界落地** — 进入魔界，享魔界寿元

## 失败后果

- **魔化失败** — 彻底消散于天地（连转世机会都没有）
- **不可逆** — 魔劫失败不可逆转
- **稀有救** — 魔帝级存在或可救，但代价极大

## 关键剧情节点

- 修仙史上成功渡魔劫者寥寥可数（< 20 位）
- 仙人叛入者多在魔劫中丧生
- 韩立：未来剧情走向（暂时正道，但存在入魔可能）

## 备注

- 魔劫是仙 / 神 / 魔三界最凶险的天劫之一
- 魔劫渡过率不足 5%
- 魔劫需「魔格」匹配才能渡过
