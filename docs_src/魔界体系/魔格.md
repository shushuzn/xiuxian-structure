# 魔格

## 定义

**魔格**是魔界修士的根本，类似于仙界的**仙职**和神界的**神格**。是魔道修炼的核心。

## 魔格等级

### 1. 凡人魔格
- **等级**：最低
- **对应**：散魔
- **来源**：凡人堕魔
- **特点**：需从零开始

### 2. 仙级魔格
- **等级**：低级
- **对应**：魔徒/魔使
- **来源**：修士叛入

### 3. 真魔格
- **等级**：中级
- **对应**：魔官/魔将
- **来源**：魔修晋升

### 4. 大魔格
- **等级**：高级
- **对应**：魔君/魔王
- **来源**：战功赫赫

### 5. 至魔格
- **等级**：极高
- **对应**：魔帝
- **来源**：统领魔界

### 6. 道魔格
- **等级**：传说
- **对应**：魔祖
- **来源**：开天辟地级

## 魔格的获得

### 1. 魔道修行
- 修炼魔功
- 积攒魔气
- 突破境界

### 2. 吞噬
- 吞噬其他魔修
- 掠夺其魔格

### 3. 天授
- 天道承认
- 自动获得高阶魔格

## 魔格的作用

### 1. 决定魔职
- 魔格等级决定可担任的魔职
- 高阶魔格可任高阶魔职

### 2. 修行速度
- 魔格越高，修行越快
- 资源获取也更多

### 3. 战力加成
- 魔格对应战力
- 魔格高者力大

## 魔格与神格

- 魔格与神格**互斥**
- 不可同时拥有
- 魔格 ↔ 神格 是天道两极
