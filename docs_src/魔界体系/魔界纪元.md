# 魔界纪元

## 定义

魔界的历史时间线——以魔帝更替或魔道大事为节点。
魔界纪元与人界 / 仙界 / 神界 / 冥界纪元不同，更暴烈、更血腥。
魔界纪元是研究魔界历史、传说、神话的基础。

## 已知纪元

### 鸿蒙纪元（最古老）

- **时间**：魔界诞生
- **特点**：魔道初现，魔界混沌
- **关键事件**：魔祖开魔辟地（传说）
- **结束**：第一批魔帝诞生

### 太古纪元

- **时间**：鸿蒙纪元后
- **特点**：魔兽化形成魔族
- **关键事件**：四大凶兽（穷奇、梼杌、饕餮、九尾魔狐）确立魔界方位
- **代表**：穷奇、梼杌、饕餮

### 远古纪元

- **时间**：太古纪元后
- **特点**：魔族繁衍，魔界秩序建立
- **关键事件**：幽都成立，罗刹 / 修罗两族崛起
- **代表**：魔帝级魔族

### 上古纪元

- **时间**：远古纪元后
- **特点**：魔族与人族混血
- **关键事件**：阿修罗族诞生，人族魔帝出现
- **代表**：阿修罗王、人族魔帝

### 中古纪元

- **时间**：上古纪元后
- **特点**：魔界扩张，与仙界 / 神界冲突加剧
- **关键事件**：第 1 位下界入魔者渡魔劫
- **代表**：入魔者先驱

### 当代纪元

- **时间**：中古纪元后
- **特点**：魔界与下界交流增加
- **关键事件**：最近 1 万年入魔者激增
- **现状**：6 方顶级魔域并存

## 关键时间点

| 时间 | 事件 |
|---|---|
| 魔界诞生 | 魔祖开魔辟地（传说）|
| 第 1 批魔帝 | 鸿蒙纪元 |
| 幽都成立 | 远古纪元 |
| 阿修罗族诞生 | 上古纪元 |
| 第 1 位下界入魔者 | 中古纪元 |
| 当代入魔潮 | 近万年 |

## 关键剧情节点

- 当代纪元（最近 1 万年）是入魔者最活跃的时代
- 当代魔帝多为阿修罗族
- 魔祖从未现身（仅在传说中）

## 备注

- 魔界纪元比神界纪元更短（魔界杀戮频繁）
- 魔界纪元常以「魔帝更替」为节点
- 当代是魔界最混乱的时代
