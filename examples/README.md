## 写作指南

想基于本知识库写更多片段、章节、剧本？建议步骤：

1. **导出数据**：`python3 scripts/export.py`，得到 `dist/xiuxian.json`
2. **喂给 LLM**：把 JSON 当参考资料，让模型「记住」所有体系细节
3. **写片段**：以「具体场景」为锚（一个坊市 / 一场战斗 / 一次突破）
4. **串联体系**：每个场景尽量覆盖 3+ 体系，避免单一设定堆砌

## 串联体系密度建议

- 短片段（~1000 字）：3-4 个体系
- 中章（~3000 字）：6-8 个体系
- 长章（~5000 字）：8-10 个体系

体系越多，越能体现「世界观厚度」，但要避免「设定说明书」式堆砌。
