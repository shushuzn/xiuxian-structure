    第一章 · 散修韩立¶

    本片段基于 xiuxian-structure 知识库（v1.0）撰写，串联 8 个体系：灵根 / 境界 / 天地灵气 / 符箓 / 丹药 / 灵石 / 势力 / 妖兽。

    一、测灵¶

    庆元镇，李家祠堂后院。

    "手放上去，不要动。"

    李叔将那块温润的测灵石按在韩立掌心。韩立只觉得掌心一阵凉意透入，像四股温热的水流沿着不同方向游走。

    测灵石亮了。

    青色。红色。黄色。白色。四色光芒交替闪烁，杂乱无序，像一锅被搅浑的稀粥。光芒持续了约三息，便暗淡下去。

    "四属性。"李叔叹了口气，拍了拍韩立瘦弱的肩膀，"伪灵根。"

    韩立其实早有预感。他爹是镇上的木匠，他娘是给李家洗衣的仆妇。爹娘都是凡人，他不知怎的竟生出了灵根——可这灵根，跟没有也没什么两样。

    伪灵根者，三四种属性混杂，修炼速度只有天灵根的十分之一，突破瓶颈的概率更是低到令人绝望。修仙界有一句刻薄话：伪灵根修仙，不如回家种田。

    李叔是散修，炼气后期，五十多岁，面容看着像三十出头。他自己没有灵根，测了三次才测出最差的废灵根——但他没放弃，靠着三十年的苦修，硬是爬到了炼气后期。整个庆元镇，能御使低阶法器飞行的也就他一人。

    "跟着我学。"李叔说，"就算修不成元婴，活个百五十年、护住这一方水土，总还做得到。"

    那年韩立七岁。

    二、坊市¶

    九年后。庆元镇外的青石坊市。

    每月逢十开市。散修、凡人商贩、低阶妖兽材料贩子在窄巷里挤作一团，空气里混着药香和血腥气。

    韩立十六岁，炼气中期。他用攒了三年的五枚下品灵石，换了一张火球符、两张金刚符、一枚遁符，外加五颗辟谷丹。

    火球符 1 枚下品，制符师是个炼气后期的老头子，一脸不耐烦地报了个数："灵力稳定，朱砂是新的，买不买？"

    韩立咬牙买下。

    辟谷丹一枚下品一颗，能顶三天不饿。他一次买五颗，是为了接下来三个月的闭关——他准备冲击炼气后期。

    掌柜是个筑基初期的中年女修，看他掏灵石时手都在抖，多看了他一眼："小兄弟，庆元镇的？"

    "是。"

    "散修？"

    "是。"

    "一个人闯荡？"

    韩立沉默了一下，点了点头。

    女修叹了口气，从柜台下摸出一只青色小瓶："这是中品符墨浸泡过的符纸，画出来的火球符威力比寻常强两成。送你了。"

    韩立一怔。

    "别谢我。"女修摆摆手，"我十六岁那年也是这样过来的。"

    三、稀薄灵气¶

    回到庆元镇外的小院，韩立盘膝坐下。

    这里的灵气是稀薄浓度——这是人界 90% 地区的常态。测灵石只微微泛光，修士吸纳一日，抵不上中型灵脉上的一刻钟。

    但他有什么办法呢。李家那条微型灵脉，核心在族长闭关的地下密室，外人一年只能借三个时辰。他这九年的修炼，一半靠那三个时辰，一半靠李叔偷偷塞给他的几枚中品灵石。

    一颗中品灵石能顶百颗下品。但他只有三颗，用一颗少一颗。

    "天灵根修炼速度是我的十倍，突破瓶颈概率是我的三倍……"韩立苦笑，"伪灵根又如何？我韩立修不成就罢了，修成了，谁还敢小看？"

    他闭上眼，开始打坐。

    吸气。呼气。灵气从四面八方涌来，但比他在李家密室里吸纳的稀薄太多。他必须把每一缕灵气都精纯入经脉，一丝都不能浪费。

    三个月。辟谷丹用完了四颗，他从炼气中期攀到了炼气中期巅峰。距离炼气后期，只差一层窗户纸。

    可是他没有丹药辅助，又没有合适的灵脉环境，那层窗户纸，他捅了三个月也没捅破。

    四、李家¶

    李家是庆元镇的小型修仙家族，族长李玄机是筑基中期，镇上唯一的"高人"。

    韩立借住在李家后院，替族中凡人子弟教些拳脚功夫，换取一日两餐和一个遮风挡雨的地方。李家不亏待他，但也不高看他——一个伪灵根的散修，在李家眼里顶多算个"半个自己人"。

    这日傍晚，韩立正在屋后练拳。

    "砰——！"

    后山方向传来巨响，伴随着低沉的嘶吼。那声音像是从地底深处传来，听得韩立头皮发麻。

    "是妖兽！"李玄机从后院飞出，御使一把青光短剑，神色凝重。

    "是那东西——火鳞蜥！炼气后期！"李家一个负责看护灵药园的炼气后期族叔，脸色惨白，"它从灵药园地下的洞穴里钻出来了！灵药园毁了三分之一！"

    火鳞蜥是低阶妖兽中最凶悍的一种。一身火红色鳞甲，刀剑难伤；口中能喷出丈许烈焰，炼气后期修士硬扛不住几下。它喜好栖息在地下洞穴，常年不出来——但不知为何，今日突然暴起。

    韩立下意识摸了摸怀里的符箓：火球符一张、金刚符两张、遁符一张。

    五、战斗¶

    火鳞蜥破土而出，三丈长的身躯横在灵药园中央，鳞片间火星四溅。

    李玄机挥剑斩去，剑光在鳞甲上划出一道白痕，却没能破防——他是筑基中期，可这头火鳞蜥是炼气后期巅峰，加上妖兽本身体魄强横，正面硬拼，他竟一时拿不下来。

    "都退后！不要靠近！"

    李家众人撤到了三十丈外，韩立也在其中。可火鳞蜥突然一甩尾，一道火焰扫向人群——

    韩立几乎是本能地激发了金刚符。

    金色护罩在韩立身上炸开。火焰撞上去，护罩颤抖了一下，没破。

    "嗯？"李玄机余光一扫，眉头微挑——这小子反应不慢。

    趁着火鳞蜥分神，李玄机御剑斜刺，刺入鳞甲缝隙，火鳞蜥痛嘶一声，转身扑向李玄机。

    "好机会！"

    韩立咬牙，掏出一张火球符，注入法力，抛向火鳞蜥后背。

    "疾！"

    火球炸开。

    火光冲天。火鳞蜥被这一下打得踉跄了一下，后背的鳞甲焦黑一片，但没死——它转头，一双竖瞳死死盯住韩立。

    "小畜生找死——！"

    火鳞蜥口中烈焰凝聚，准备给韩立再来一下。

    韩立手中捏着遁符，紧张得手心冒汗。等烈焰喷出的一瞬间——

    "起！"

    他激发遁符，身形瞬间出现在十丈之外。烈焰落空，灼烧了身后一片树木。

    "漂亮！"李玄机大笑，"好小子！"

    他抓住火鳞蜥分神的这一瞬，手中短剑化作十丈青虹，猛然一斩——"噗"的一声，火鳞蜥的头颅飞起。

    六、离去¶

    火鳞蜥的尸体被抬到李家祠堂前，族中老幼围观议论。

    "这一剑劈得好！"

    "听说是族长一个人杀的。"

    "那小子也帮了忙。"

    韩立远远站在人群外，看着李玄机被族人簇拥的背影，摸了摸怀中空了一半的符箓口袋。

    他心里明白：自己的"帮忙"，不过是火上浇了点油。真正决定胜负的，是李玄机那一剑——筑基中期对炼气后期妖兽的绝对碾压。

    可这世间，又有几个李玄机呢？

    他想起了李叔当年的话：「伪灵根修仙，不如回家种田。」

    种田？庆元镇的田亩已经被妖兽糟蹋了三分之一。他不修仙，这镇上能挡得住下一头妖兽的，又有几人？

    这一晚，韩立收拾了简单的行囊。

    他要去外面走走。听说北荒深处有一处中型灵脉的散修聚集地，叫"万灵山坊市"，专门收留像他这样的野修。那里或许能让他在寿元耗尽之前，触摸到那层窗户纸。

    临行前，他去祠堂后院给测灵石磕了三个头。

    青色。红色。黄色。白色。

    四色杂光依然杂乱无序。

    但那又如何？

    （全文完）

    写作后记 · 涉及的体系¶

    本片段刻意串联了知识库中的 8 个体系：

    章节 体系 关键设定
    一、测灵 灵根 伪灵根 = 4 属性混杂，速度 × 1.0，突破 × 0.3
    二、坊市 灵石 / 符箓 / 丹药 下品灵石购买力 / 火球符 / 辟谷丹
    三、稀薄灵气 天地灵气 / 境界 稀薄灵气区修炼效率 / 炼气中期到瓶颈
    四、李家 势力 / 妖兽 修仙家族 / 火鳞蜥（炼气后期）
    五、战斗 符箓 / 境界 金刚符（防御）/ 火球符（攻击）/ 遁符（逃命）
    六、离去 势力 / 灵脉 散修联盟 / 中型灵脉吸引力

    整篇不出现任何"知识库式"的硬解释，全部内化在叙事中。
