# 冥兽

## 定义

冥界特有的高阶生物——实力远超仙界灵兽，部分甚至超过鬼修。
冥兽是冥道的宠儿，拥有独特的天赋神通（死寂、轮回、吞噬魂魄）。
冥兽与鬼修可签订契约（参考 [契约体系](../契约体系/契约.md)），也可被鬼修收服。

## 等级

| 等级 | 实力 | 代表 |
|---|---|---|
| 凡兽 | 凡人级 | 冥界低阶生物 |
| 灵兽 | 仙人级 | 普通冥兽 |
| 冥兽 | 仙人级以上 | 高阶冥兽 |
| 圣兽 | 鬼君级 | 顶级冥兽 |
| 冥兽王 | 鬼王级 | 传说冥兽 |
| 道兽 | 冥帝级 | 传说级 |

## 代表冥兽

### 上古冥兽（顶级）

- **谛听** — 冥兽王级，聆听万物（地藏王坐骑）
- **毕方冥种** — 鬼王级，阴火之鸟
- **九幽冥蛇** — 鬼君级，剧毒无双
- **黄泉冥蝶** — 鬼君级，幻术无双
- **忘川冥鱼** — 鬼君级，忘川之主
- **三足冥鸦** — 鬼王级，死亡预言

### 普通冥兽

- **幽冥狼** — 鬼使级，群居
- **噬魂蝶** — 鬼官级，吸取魂魄
- **冥蝙蝠** — 鬼使级，群居

## 冥兽与人

### 签约

- 冥兽可与鬼修签订契约（参考 [契约体系](../契约体系/契约.md)）
- 多采用「魂契」
- 「主从契约」极常见（冥兽比神兽更易臣服）

### 化形

- 冥兽可化为人形
- 化形后保留部分冥兽特征（如冥龙角）
- 化形冥兽可参与冥界政治

### 战斗

- 冥兽战斗力极强
- 部分冥兽可与鬼王单挑
- 顶级冥兽需冥帝出手

## 关键剧情节点

- 谛听是冥界四大圣兽之首
- 黄泉冥蝶是黄泉的守护冥兽
- 忘川冥鱼是忘川河的主宰

## 备注

- 冥兽比神兽更神秘（与轮回相关）
- 冥兽驯服率比神兽高
- 谛听是传说级，仅冥帝可见真身
