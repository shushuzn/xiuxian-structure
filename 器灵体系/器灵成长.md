# 器灵成长

## 定义

器灵通过**滋养、战斗、机缘**逐渐强大、升级的过程。
器灵成长是法器从"法宝"进阶为"灵宝"乃至"仙器"的唯一路径——没有成长的器灵，永远只是雏形。
器灵成长速度极慢，通常以**百年**为单位计量。

## 成长机制

### 滋养成长（基础）

- **天材地宝滋养** — 主人定期以灵物喂养器灵
  - 千年灵药 → 雏形 → 小成
  - 万年灵药 → 小成 → 完整
  - 仙品灵物 → 完整 → 大成
- **神识温养** — 主人定期以神识与器灵"对话"、注入神识
- **灵气浸泡** — 将法器长期放置于灵脉之上

### 战斗成长（加速）

- **生死之战** — 法器在生死大战中激发潜能，器灵加速成长
- **主人渡劫淬炼** — 主人渡劫时法器一同受劫雷淬炼
  - 雷劫可催生器灵雏形
  - 高阶雷劫可让器灵跃升数级
  - 参考 [雷劫淬体](../雷劫体系/雷劫淬体.md)
- **器灵觉醒** — 关键时刻器灵自主觉醒（如主人濒死时）

### 主人突破带动

- 主人突破大境界时，神识与法力质变，可带动器灵同步成长
- 主人从元婴突破化神时，器灵易从雏形跃至完整

### 特殊机缘

- **吸收妖丹** — 吞噬化形期以上妖丹可大幅促进器灵成长（邪法）
- **同源法器融合** — 两件同源法器可尝试融合（成功率极低）
- **天劫特殊赐予** — 极少数修士渡劫时获得天劫"赐灵"，器灵一夜大成（传说级）

## 成长阶段（典型时间表）

| 阶段 | 时间（自愿认主 + 持续滋养） | 关键事件 |
|---|---|---|
| 雏形 | 0 | 器灵诞生 |
| 小成 | 100-500 年 | 主人元婴期突破 |
| 完整 | 500-2000 年 | 主人化神期突破，或雷劫淬炼 |
| 大成 | 2000-5000 年 | 多次雷劫淬炼 + 长期滋养 |
| 至臻 | 10000+ 年 | 飞升仙界后 |

## 成长的风险

### 失控成长

- 器灵成长过快（血祭邪法催生）→ 器灵反噬概率大增
- 主人实力跟不上器灵成长 → 器灵可能"看不起"主人

### 停滞成长

- 器灵长期不成长 → 关系疏远，易反噬
- 主人长期闭关不与器灵沟通 → 器灵"沉睡"

### 夭折

- 滋养不当（用了相克灵物）→ 器灵意识消散
- 主人死亡时器灵未达大成 → 器灵随主人一同消散

## 加速成长的禁忌方法

### 血祭催灵（邪法）

- **机制**：以修士精血或魂魄强行灌注法器，催生器灵
- **代价**：被灌注者必死，反噬概率 80%+
- **后果**：器灵多为凶物，无完整意识
- **稀有度**：魔修邪修所用，正道不齿

### 夺舍器灵（极邪）

- **机制**：将活人神魂强行炼入法器，作为"器灵"
- **代价**：被夺舍者求生不得求死不能
- **后果**：器灵有完整人智，但充满怨念，反噬率 100%
- **稀有度**：传说级邪法，几无人敢用

### 天劫强催（极险）

- **机制**：在渡劫时以秘法让器灵强行吸收劫雷
- **代价**：器灵可能被劫雷直接抹杀
- **后果**：成功则器灵一夜数级，失败则形神俱灭
- **稀有度**：极少见，是传说中"雷师"等大能所用的方法

## 关键剧情节点

- **雷师印**：雷师本人在世时，雷师印器灵已达大成；雷师死后器灵沉睡了 7000 年，被韩立接受后经长期滋养重新觉醒
- **养魂木**：养魂木中的南陇侯残魂（本质为"残魂器灵"）经 80 年温养仍未完全苏醒，需九幽还魂草加速
