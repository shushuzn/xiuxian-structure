# 化形期（妖修境界）

## 定位
妖修突破"兽形"的关键境界，对应人界修士的**结丹期**。
化形成功后的妖修可融入人界社会，开启真正的修仙之路。

## 等级划分

| 化形等级 | 对应人界境界 | 实力对比 |
|---|---|---|
| **化形初期** | **结丹期** | 结丹级 |
| **化形中期** | **元婴期** | 元婴级 |
| **化形后期** | **化神期** | 化神级 |
| **化形巅峰** | **化神后期** | 化神巅峰 |
| **渡劫化形** | **渡劫期** | 渡劫级 |

## 寿命上限
- **化形初期**：约 800 年（与人界结丹期相当）
- **化形中期**：约 1500 年（与人界元婴期相当）
- **化形后期**：约 3000 年（与人界化神期相当）

## 化形条件

### 必备条件
- **修为**：达到七级（结丹级）
- **丹药**：化形丹（成功率 50-70%）
- **灵气**：需在灵脉之上化形
- **时间**：化形过程需 7-49 日，期间极度脆弱

### 化形丹
- **品级**：宝品
- **主药**：千年灵药 + 妖修精血
- **辅药**：数味百年灵药
- **炼制**：需元婴期炼丹师

### 化形失败
- 退回原形
- 重伤需 100 年恢复
- 二次化形成功率减半

## 化形后

### 形态
- **完全人形**：可融入人界社会
- **半人形**：保留原特征（如蛟龙保留角、蛇妖保留竖瞳）
- **原形**：化形不完全，可强行变回原形战斗

### 能力变化
- **神识**：与人界同阶相当
- **法力**：与人界同阶相当
- **天赋**：保留原妖兽天赋（如蛇妖寒气、蛟龙水系）
- **修炼**：可修习人族功法 + 妖修功法

### 限制
- 化形后**生育能力下降**（妖修与人族通婚困难）
- **天道压制**：妖修渡劫比人界修士更严苛
- **文化隔阂**：与人界修士存在天然隔阂

## 修炼路径

```
未化形妖修（练气~筑基）
   ↓ 修炼百年 + 化形丹
化形初期（结丹级）
   ↓ 500 年修炼 + 长老认可
化形中期（元婴级）
   ↓ 千年修炼
化形后期
```

## 著名化形妖修
- **潮汐老祖**（化形后期）：潮汐宫宫主，实力堪比元婴中期
- **九幽寒蟒**（化形初期）：aoyue 本命灵兽
- **云渡剑鹤**（筑基后期）：南陇侯饲养之灵兽

- **vol2**：韩立需从潮汐宫获得化形期妖丹续养魂木
