# 心魔劫

## 定义

**心魔劫**是修士在突破大境界时，因**心魔**引发的心境考验。是渡劫期最凶险的劫难之一。

## 心魔劫的触发

- **境界**：大境界突破时（结丹/元婴/化神/炼虚/合体）
- **心魔**：平时积压的心魔在此时爆发
- **类型**：贪/嗔/痴/执念/欲/恶/悟等

## 心魔劫的过程

### 1. 心境触发
- 修士闭关突破
- 心魔化形
- 与本心对抗

### 2. 心境考验
- 心魔模拟各种诱惑
- 修士须坚定本心
- 否则被心魔吞噬

### 3. 心境顿悟
- 战胜心魔者心境升华
- 修为大进
- 获得「渡心」之名

## 心魔劫的分类

| 心魔 | 描述 |
|---|---|
| 贪魔劫 | 贪恋权财 |
| 嗔魔劫 | 怒火攻心 |
| 痴魔劫 | 痴迷大道 |
| 执念劫 | 执念放不下 |
| 欲魔劫 | 欲望缠身 |
| 恶魔劫 | 恶念作祟 |
| 悟魔劫 | 假悟实迷 |

## 应对方法

### 1. 丹药
- 清心丹
- 凝神丹
- 护神丹

### 2. 阵法
- 渡劫阵
- 清心阵

### 3. 心境修为
- 平时修心
- 积累功德
- 减少心魔
