# 神通

## 概述

神通是修仙者通过灵力运转、法则领悟而施展的进攻型法术技能。神通的品级决定其威力的上限，但实际发挥受修者境界、法力储量与神识强度的影响。高品级神通往往需要对应境界才能驱动，越级施展可能导致法力枯竭或反噬。每个修者通常会修炼一门本命神通作为压轴手段，配合常规神通形成战斗体系。

## 等级/分类

神通按品级由低到高分为七档：

- **黄级** — 炼气期可用，基础法术，威力有限，如火焰拳
- **玄级** — 筑基期可用，附带属性特效，如寒冰掌可冻结经脉
- **地级** — 结丹期可用，引动天地灵气，如紫电神雷
- **天级** — 元婴期可用，涉及法则雏形，如乾坤一指破万法
- **仙级** — 化神期及以上可用，法则级攻击，如万象天印封印百里
- **神级** — 仙人境及以上可用，规则级攻击，如诛神剑诀斩灭神魂
- **圣级** — 神帝级可用，灭世级威力，如灭世天威覆盖整个界面

## 效果/特性

神通的实战特性包括：

- **属性相克** — 金属性克木、木克土、土克水、水克火、火克金，循环相克
- **消耗随品级指数增长** — 仙级神通消耗法力是天级的十倍以上
- **领悟门槛** — 高阶神通需对应法则感悟，不可仅凭境界强行修炼
- **组合释放** — 部分神通可组合成连招，威力相乘而非相加
- **器灵增幅** — 拥有器灵的法器可增幅神通威力，增幅幅度与器灵等级正相关
- **神通共鸣** — 高阶神通释放时可能引发天地异象，吸引周围修士注意

## 修炼方法

神通的修炼通常分为三个阶段：

- **入门** — 理解神通运行原理，可勉强释放，威力不足三成
- **小成** — 熟练运转，可正常释放，威力可达五至七成
- **大成** — 与自身道法融合，瞬发且威力达到十成，可应对各种战场变化
- **圆满** — 神通化入本能，可随意变招组合，形成个人独特的战斗风格

## 神通搭配

合理的神通搭配可大幅提升战斗效率：

- **连击搭配** — 以控制型神通起手，接爆发型神通输出，最后以终结型神通收尾
- **属性循环** — 利用五行相克原理连续切换属性神通，使对手防不胜防
- **攻防一体** — 同时修炼攻击与防御类神通，确保任何战况下都有应对手段
- **范围与单体** — 搭配群攻与单体爆发神通，兼顾清场与斩首
