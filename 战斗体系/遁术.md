# 遁术

## 概述

遁术是修仙者利用五行元素进行短距离移动的实用法术。遁术的施展速度与隐蔽性取决于修者与对应元素的亲和度以及功法属性匹配度。五行遁术是低阶修士最常用的移动与规避手段，在战斗中可用于突袭、撤退与闪避。大多数修士至少精通两种遁术以应对不同战斗环境。

## 等级/分类

遁术按五行属性分为七大类，各具特点：

- **金遁** — 中等难度，速度极快，需借助金属性物质为媒介，在矿脉或法器密集处效果最佳
- **木遁** — 低等难度，速度中等，在丛林环境中效果最佳，城内或荒漠难以施展
- **水遁** — 低等难度，速度中等，化水而遁，无声无息，是暗杀与隐蔽行动的优选
- **火遁** — 中等难度，速度快，但遁走时有火光，隐蔽性差，适合正面突袭
- **土遁** — 低等难度，速度慢，最隐蔽，适合伏击撤退，可在地下潜行
- **风遁** — 高等难度，速度极快，无形无影，高阶修士首选，无视大部分地形
- **雷遁** — 高等难度，瞬间移动，消耗巨大，使用后需要短暂恢复

## 效果/特性

遁术实战应用要点：

- **元素媒介依赖** — 遁术施展需要在对应元素环境中或有对应媒介，无水之地无法用水遁
- **神识锁定对抗** — 高阶修士可用神识锁定遁术轨迹，在遁术未完成时发动打断
- **遁术与阵法结合** — 在阵法范围内可有"阵遁"效果，无视元素限制
- **风雷遁术的优势** — 风遁与雷遁不依赖固定元素媒介，适用场景更广
- **遁术等级与境界挂钩** — 高难度遁术需要足够修为支撑，强行施展易遭反噬
- **五行遁术互克** — 土遁克水遁、水遁克火遁，属性克制同样适用于遁术对抗

## 修炼要点

遁术修炼的注意事项：

- **元素亲和度** — 与灵根属性匹配的遁术修炼速度更快，效果更好
- **持久练习** — 遁术需要大量实战练习才能在战斗中熟练运用
- **神识辅助** — 遁术过程中的方向判断与落点选择依赖神识定位
- **环境适应** — 建议在不同地形环境中练习同一遁术以适应各种战场

## 遁术战斗运用

遁术在不同战斗场景中的运用技巧：

- **突袭** — 以雷遁或风遁快速接近敌人发动攻击，打对手一个措手不及
- **撤退** — 以土遁或水遁隐蔽撤离，避免被对手追踪
- **闪避** — 以瞬发遁术规避致命攻击，与神识预判配合效果更佳
- **声东击西** — 以火遁制造动静吸引注意，配合其它遁术从另一方向突袭
