# 神兽

## 定义

神界特有的高阶生物——实力远超仙界灵兽，部分甚至超过神人。
神兽是天道的宠儿，拥有独特的天赋神通。
神兽与神人可签订契约（参考 [飞升体系 - 契约](../契约体系/契约.md)），也可被神人收服。

## 等级

| 等级 | 实力 | 代表 |
|---|---|---|
| 凡兽 | 凡人级 | 神界低阶生物 |
| 灵兽 | 仙人级 | 普通神兽 |
| 神兽 | 仙人级以上 | 高阶神兽 |
| 圣兽 | 神君级 | 顶级神兽 |
| 神兽王 | 神王级 | 传说神兽 |
| 道兽 | 神帝级 | 传说级 |

## 代表神兽

### 四方神兽（传说级）

- **青龙** — 掌东方，神帝级
- **白虎** — 掌西方，神帝级
- **朱雀** — 掌南方，神帝级
- **玄武** — 掌北方，神帝级

### 上古神兽（顶级）

- **鲲鹏** — 神王级，扶摇九天
- **烛龙** — 神王级，睁眼为昼闭眼为夜
- **应龙** — 神王级，掌控水神力

### 普通神兽

- **九尾狐** — 神官级，幻术无双
- **毕方** — 神官级，火神力
- **穷奇** — 神将级，凶兽

## 神兽与人

### 签约

- 神兽可与神人签订契约（参考 [契约体系](../契约体系/契约.md)）
- 多采用「平等契约」或「魂契」
- 「主从契约」极少（神兽极少臣服）

### 化形

- 神兽可化为人形
- 化形后保留部分神兽特征（如青龙鳞）
- 化形神兽可参与神界政治

### 战斗

- 神兽战斗力极强
- 部分神兽可与神王单挑
- 顶级神兽需神帝出手
