# 神气

## 定义

神界的基础资源——比仙气更高级的天地精华，是神人修炼、生存的根本。
神气浓度是仙气的百倍以上，蕴含完整的天地法则。
神气是神界一切力量的来源，没有神气的神人无法施展任何神术。

## 等级

| 浓度 | 等级 | 适用 |
|---|---|---|
| 稀薄神气 | 1 | 神民 |
| 普通神气 | 2 | 神徒 / 神使 |
| 浓郁神气 | 3 | 神官 / 神将 |
| 化液神气 | 4 | 神君 / 神王 |
| 神气精华 | 5 | 神尊 / 神帝 |
| 道气 | 传说级 | 道祖 |

## 与仙气的差异

| 维度 | 仙气 | 神气 |
|---|---|---|
| 浓度 | 百倍于人界灵气 | 百倍于仙气 |
| 法则 | 部分显化 | 完全显化 |
| 使用 | 仙人施展法术 | 神人施展神术 |
| 跨界 | 不可 | 不可 |

## 来源

- **神脉** — 神界特有的神脉（比仙脉更高级）
- **天道自然凝结** — 神气浓郁处自然凝结（耗时千年）
- **神人炼制** — 高阶神人可炼制神气精华
- **跨界偷渡** — 传说中存在（被发现必死）

## 用途

- **修炼** — 神人日常消耗神气（普通神人每日消耗 1 单位浓郁神气）
- **突破** — 神界突破消耗极大（数百万单位）
- **战斗** — 神术以神气为能量
- **炼器** — 炼制神器
- **炼丹** — 炼制神丹

## 限制

- **跨界不可用** — 神气仅限神界
- **不可量产** — 神脉资源有限
- **因果关联** — 大量使用神气加重因果
