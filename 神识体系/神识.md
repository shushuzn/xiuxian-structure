# 神识

## 定义

修士的**精神力外延**，是感知外界、操控法器、布阵炼丹、抵御心魔与识战的底层能力。
神识随境界跃升而发生**质变**——不仅是范围扩大，更是从"模糊感知"到"离体夺舍"的根本性升级。
神识的强度直接决定修士在战斗、炼丹、布阵、符箓等多个维度的上限。

## 构成要素

### 本质

- **神魂的外显** — 神识是神魂的延伸，神魂强大则神识强大
- **可锻炼** — 通过修炼神识类功法、服用神识类丹药可临时或永久增强
- **可攻击** — 高阶神识可化为识刺、识爆、识域，对敌方神魂造成直接伤害
- **可附身** — 元婴期以上神识可短暂离体附身低阶修士或凡人

### 范围公式

神识范围与境界正相关，并受以下因素影响：

- **基础范围** = 境界决定的范围（见各境界 .md）
- **神识类功法加成** = +50% ~ +200%
- **神识类丹药** = 临时 +30% ~ +100%（有时限）
- **灵气浓度** = 浓郁灵气环境下 +20%
- **肉身状态** = 重伤或疲惫时 -50% ~ -80%

### 等级与质变

- **微神识**（[炼气期](炼气神识.md)）— 模糊感知数丈，无法离体
- **小神识**（[筑基期](筑基神识.md)）— 清晰数十丈，可驱动符箓
- **大神识**（[结丹期](结丹神识.md)）— 可穿墙透视，可短暂附身低阶修士
- **元神识**（[元婴期](元婴神识.md)）— 离体、夺舍、识战，神魂具象化
- **法神识**（化神期及以上）— 可触及天地法则层面，留意万里外异动

### 用途

- **探查** — 感知范围内一切动静，可穿透障碍物（高阶神识）
- **战斗** — 驱动法器、施展法术、[神识攻击](神识攻击.md)
- **炼丹** — 感知火候、药材融合状态（必需）
- **布阵** — 引导阵法灵力走向（必需）
- **画符** — 封入神识印记激活符箓（必需）
- **认主** — 与法器器灵沟通、留下神识印记
- **传讯** — 通过传讯符/玉简传递神识片段
- **抵御心魔** — 神识强大者更难被心魔攻破
