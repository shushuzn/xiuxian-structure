# 关联索引

## 灵根 → 功法
灵根属性决定可修行功法的属性方向：
- 金灵根 → 金属性功法
- 木灵根 → 木属性功法
- 变异灵根 → 变异属性功法
- 灵根品级越低，对功法品级的适配上限越低

## 灵根 → 境界
灵根品级直接影响修炼速度和突破瓶颈概率：
- 天灵根突破瓶颈天生占优
- 伪灵根每突破一个境界都比天灵根难数倍

## 灵气 → 境界
灵气浓度决定修炼效率上限：
- 稀薄灵气下炼气期尚可，筑基困难
- 突破大境界（炼气→筑基）需要大量灵气灌体

## 灵气 → 灵脉 → 势力
灵脉等级决定宗门能供养多少高阶修士：
- 微型灵脉 → 散修或小家族
- 大型灵脉 → 大宗门根基

## 灵脉 → 灵石
灵脉产出灵石，是灵石的主要来源：
- 小型灵脉年产下品灵石数百
- 大型灵脉可产出中品甚至上品灵石

## 灵石 → 势力经济
灵石是势力运转的经济基础：
- 弟子俸禄、悬赏任务、贸易往来均以灵石结算
- 势力规模与灵石储备正相关

## 功法 → 境界
功法品级决定修炼上限：
- 黄级功法最高到筑基
- 玄级可到结丹
- 地级可到元婴
- 天级才能走得更远

## 丹药 → 境界
丹药品级对应可使用的境界：
- 凡品丹药 → 炼气期
- 灵品丹药 → 筑基期
- 宝品丹药 → 结丹期
- 仙品丹药 → 元婴期及以上

## 丹药 → 突破
特定丹药可辅助突破境界瓶颈：
- 筑基丹 → 炼气突破筑基
- 结丹丹 → 筑基突破结丹
- 无丹突破成功率极低，高阶突破几乎不可能

## 丹药 → 灵物
丹药以天地灵物为原料：
- 灵药品级决定丹药品级上限
- 稀有灵物对应稀有丹方

## 妖兽 → 势力
顶级妖兽可建立妖修势力：
- 元婴期妖王可建立大型妖修势力
- 化神期妖皇可与顶级宗门分庭抗礼

## 妖兽 → 资源
妖兽是重要的资源来源：
- 内丹可入药、炼器
- 皮毛、兽骨可炼器
- 灵兽可作为修士伙伴

## 阵法 → 势力防御
大型势力常设护山大阵：
- 宗门护山大阵是最后防线
- 阵法消耗灵石作为能量源
- 高阶阵法可与元婴期修士抗衡

## 符箓 → 修为补足
符箓弥补低阶修士法力不足：
- 炼气期可购买筑基级符箓临时使用
- 符箓是一次性消耗品
- 与阵法功能重叠但更便携

## 灵根 → 副业
灵根属性决定修士适合的副业方向：
- 金属性 → 炼器（淬炼金属）
- 木属性 → 炼丹、灵植
- 水属性 → 阵法、符箓
- 火属性 → 炼丹、炼器
- 土属性 → 炼体、阵法
- 雷/冰/风变异 → 战斗专精，副业较弱

## 灵根 → 法器契合
灵根属性决定法器威力加成：
- 金属性飞剑 + 金属性天灵根 = 威力 +30%
- 火属性法宝 + 火属性天灵根 = 威力 +30%
- 属性相克时不可用法器（火灵根不可用纯水属性法器）
- 变异灵根与同源五行法器有部分通用性（雷灵根可用金飞剑 +15%）

## 灵气浓度 → 境界
灵气浓度直接决定可修炼境界上限：
- 稀薄灵气仅够炼气期，突破筑基困难（成功率 < 5%）
- 普通灵气可支撑筑基期，结丹需灵脉辅助
- 浓郁灵气适合结丹期长期修炼
- 化液灵气是元婴/化神期修士梦寐以求的修炼环境

## 灵脉 → 势力规模
灵脉等级直接决定势力规模上限：
- 微型灵脉仅能支撑散修或小家族
- 小型灵脉是小型宗门根基
- 中型灵脉供养 3-10 位结丹期修士
- 大型灵脉供养元婴期 + 数十结丹期
- 超级灵脉是化神期修士修炼圣地

## 散修 → 功法
散修最常见的"技能树"是自创或交易获得功法：
- 黄级功法易得（坊市散修摆摊）
- 玄级功法需小型宗门或家族传承
- 地级及以上功法几乎只有宗门传承
- 散修常因缺少高级功法而止步筑基

## 阵法 → 势力经济
护山大阵是中型以上宗门的必备支出：
- 中型宗门护山大阵年耗中品灵石 50-200 枚
- 大型宗门护山大阵年耗上品灵石 10-50 枚
- 阵法维护占势力年支出 10%-30%
- 阵法等级直接影响势力防御能力

## 符箓 → 修士等级差弥补
符箓是低阶修士"越级战斗"的关键：
- 炼气期持筑基级符箓可短暂对抗筑基期
- 筑基期持结丹级符箓可作为底牌
- 散修常备攻击/防御/遁符三件套
- 高阶符箓是战略物资，宗门严格管控
