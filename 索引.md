# 关联索引

## 灵根 → 功法
灵根属性决定可修行功法的属性方向：
- 金灵根 → 金属性功法
- 木灵根 → 木属性功法
- 变异灵根 → 变异属性功法
- 灵根品级越低，对功法品级的适配上限越低

## 灵根 → 境界
灵根品级直接影响修炼速度和突破瓶颈概率：
- 天灵根突破瓶颈天生占优
- 伪灵根每突破一个境界都比天灵根难数倍

## 灵气 → 境界
灵气浓度决定修炼效率上限：
- 稀薄灵气下炼气期尚可，筑基困难
- 突破大境界（炼气→筑基）需要大量灵气灌体

## 灵气 → 灵脉 → 势力
灵脉等级决定宗门能供养多少高阶修士：
- 微型灵脉 → 散修或小家族
- 大型灵脉 → 大宗门根基

## 功法 → 境界
功法品级决定修炼上限：
- 黄级功法最高到筑基
- 玄级可到结丹
- 地级可到元婴
- 天级才能走得更远

## 丹药 → 境界
丹药品级对应可使用的境界：
- 凡品丹药 → 炼气期
- 灵品丹药 → 筑基期
- 宝品丹药 → 结丹期
- 仙品丹药 → 元婴期及以上

## 丹药 → 突破
特定丹药可辅助突破境界瓶颈：
- 筑基丹 → 炼气突破筑基
- 结丹丹 → 筑基突破结丹
- 无丹突破成功率极低，高阶突破几乎不可能

## 丹药 → 灵物
丹药以天地灵物为原料：
- 灵药品级决定丹药品级上限
- 稀有灵物对应稀有丹方
