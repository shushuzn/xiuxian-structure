    关联索引¶

    📖 概念速查：见 概念汇总 — 全局术语表（境界 / 法器 / 器灵 / 契约 / 神识 / 心魔 / 雷劫 / 飞升 / 仙界 / 神界 / 魔界 / 冥界 等核心概念的统一定义）

    灵根 → 功法¶

    灵根属性决定可修行功法的属性方向： - 金灵根 → 金属性功法 - 木灵根 → 木属性功法 - 变异灵根 → 变异属性功法 - 灵根品级越低，对功法品级的适配上限越低

    灵根 → 境界¶

    灵根品级直接影响修炼速度和突破瓶颈概率： - 天灵根突破瓶颈天生占优 - 伪灵根每突破一个境界都比天灵根难数倍

    灵气 → 境界¶

    灵气浓度决定修炼效率上限： - 稀薄灵气下炼气期尚可，筑基困难 - 突破大境界（炼气→筑基）需要大量灵气灌体

    灵气 → 灵脉 → 势力¶

    灵脉等级决定宗门能供养多少高阶修士： - 微型灵脉 → 散修或小家族 - 大型灵脉 → 大宗门根基

    灵脉 → 灵石¶

    灵脉产出灵石，是灵石的主要来源： - 小型灵脉年产下品灵石数百 - 大型灵脉可产出中品甚至上品灵石

    灵石 → 势力经济¶

    灵石是势力运转的经济基础： - 弟子俸禄、悬赏任务、贸易往来均以灵石结算 - 势力规模与灵石储备正相关

    功法 → 境界¶

    功法品级决定修炼上限： - 黄级功法最高到筑基 - 玄级可到结丹 - 地级可到元婴 - 天级才能走得更远

    丹药 → 境界¶

    丹药品级对应可使用的境界： - 凡品丹药 → 炼气期 - 灵品丹药 → 筑基期 - 宝品丹药 → 结丹期 - 仙品丹药 → 元婴期及以上

    丹药 → 突破¶

    特定丹药可辅助突破境界瓶颈： - 筑基丹 → 炼气突破筑基 - 结丹丹 → 筑基突破结丹 - 无丹突破成功率极低，高阶突破几乎不可能

    丹药 → 灵物¶

    丹药以天地灵物为原料： - 灵药品级决定丹药品级上限 - 稀有灵物对应稀有丹方

    妖兽 → 势力¶

    顶级妖兽可建立妖修势力： - 元婴期妖王可建立大型妖修势力 - 化神期妖皇可与顶级宗门分庭抗礼

    妖兽 → 资源¶

    妖兽是重要的资源来源： - 内丹可入药、炼器 - 皮毛、兽骨可炼器 - 灵兽可作为修士伙伴

    阵法 → 势力防御¶

    大型势力常设护山大阵： - 宗门护山大阵是最后防线 - 阵法消耗灵石作为能量源 - 高阶阵法可与元婴期修士抗衡

    符箓 → 修为补足¶

    符箓弥补低阶修士法力不足： - 炼气期可购买筑基级符箓临时使用 - 符箓是一次性消耗品 - 与阵法功能重叠但更便携

    灵根 → 副业¶

    灵根属性决定修士适合的副业方向： - 金属性 → 炼器（淬炼金属） - 木属性 → 炼丹、灵植 - 水属性 → 阵法、符箓 - 火属性 → 炼丹、炼器 - 土属性 → 炼体、阵法 - 雷/冰/风变异 → 战斗专精，副业较弱

    灵根 → 法器契合¶

    灵根属性决定法器威力加成： - 金属性飞剑 + 金属性天灵根 = 威力 +30% - 火属性法宝 + 火属性天灵根 = 威力 +30% - 属性相克时不可用法器（火灵根不可用纯水属性法器） - 变异灵根与同源五行法器有部分通用性（雷灵根可用金飞剑 +15%）

    灵气浓度 → 境界¶

    灵气浓度直接决定可修炼境界上限： - 稀薄灵气仅够炼气期，突破筑基困难（成功率 < 5%） - 普通灵气可支撑筑基期，结丹需灵脉辅助 - 浓郁灵气适合结丹期长期修炼 - 化液灵气是元婴/化神期修士梦寐以求的修炼环境

    灵脉 → 势力规模¶

    灵脉等级直接决定势力规模上限： - 微型灵脉仅能支撑散修或小家族 - 小型灵脉是小型宗门根基 - 中型灵脉供养 3-10 位结丹期修士 - 大型灵脉供养元婴期 + 数十结丹期 - 超级灵脉是化神期修士修炼圣地

    散修 → 功法¶

    散修最常见的"技能树"是自创或交易获得功法： - 黄级功法易得（坊市散修摆摊） - 玄级功法需小型宗门或家族传承 - 地级及以上功法几乎只有宗门传承 - 散修常因缺少高级功法而止步筑基

    阵法 → 势力经济¶

    护山大阵是中型以上宗门的必备支出： - 中型宗门护山大阵年耗中品灵石 50-200 枚 - 大型宗门护山大阵年耗上品灵石 10-50 枚 - 阵法维护占势力年支出 10%-30% - 阵法等级直接影响势力防御能力

    符箓 → 修士等级差弥补¶

    符箓是低阶修士"越级战斗"的关键： - 炼气期持筑基级符箓可短暂对抗筑基期 - 筑基期持结丹级符箓可作为底牌 - 散修常备攻击/防御/遁符三件套 - 高阶符箓是战略物资，宗门严格管控

    知识库 → 创作示例¶

    知识库产出的实际作品： - 小说：第一章·散修韩立 — 串联 8 体系 - 互动：测灵殿的选择 — 状态机驱动 - LLM 节点：夜探禁地 — generate_node.py 生成示例 - 数据：xiuxian.json — 整本知识库

    突破 → 心魔 + 雷劫（v1.5 新增）¶

    所有大境界突破皆承受两重考验，常并发： - 境界越高 → 雷劫越强 → 心魔越烈 - 失败后果：散功（轻则失修为、重则形神俱灭） - 成功收益：雷劫淬体（肉身 / 神魂 / 法力三重升华） - 应对工具：清心丹（抗心魔）· 凝神丹（稳神魂）· 渡劫丹（抗雷劫）· 渡劫阵（综合辅助）

    具体对应： - 结丹期 → 元婴期：三九天劫（9 道）+ 贪魔/执念魔 - 元婴期 → 化神期：六九天劫（18 道）+ 嗔魔/痴魔 - 化神期 → 炼虚期：同上 - 炼虚期 → 合体期：九九天劫（27 道）+ 嗔魔/痴魔 - 合体期 → 渡劫期：同上 + 所有心魔 - 渡劫期 → 大乘期：九九天劫飞升之劫 + 所有心魔至极

    境界 → 神识（v1.6 新增）¶

    神识随境界跃升而质变，是串联阵法/符箓/丹药/法器的底层能力： - 炼气期 → 微神识（3-10 丈，模糊感知） - 筑基期 → 小神识（30-100 丈，可画符/炼丹） - 结丹期 → 大神识（100-500 丈，可穿墙、可附身） - 元婴期 → 元神识（数里-数十里，可离体、可识战、可夺舍）

    神识攻击 是元婴期以上修士的标志性作战方式（识刺/识爆/识域/识灭）。

    法器 → 器灵（v1.7 新增）¶

    法宝级及以上的法器可蕴含器灵——法器内部的意识体： - 法宝 → 器灵雏形（微弱意识，仅能传信） - 灵宝 → 器灵完整（可沟通、部分自主行动） - 传说级 / 仙器 → 器灵大成 / 至臻（心意相通 / 灵魂融合）

    四个核心机制： - 器灵认主 — 自愿 / 仪式 / 强迫 / 平等（4 种） - 器灵沟通 — 警示 / 单向传信 / 神识对话 / 心意相通 / 灵魂融合（5 种） - 器灵反噬 — 轻 / 中 / 重 / 完全（4 档） - 器灵成长 — 自然 / 天材 / 雷劫 / 战斗 / 血祭（5 种）

    灵兽 / 势力 → 契约（v1.7 新增）¶

    修士与灵兽、妖修、人、法器、势力之间的灵魂绑定关系，4 类契约由浅到深： - 主从契约 — 强者压制弱者（灵兽袋基础） - 平等契约 — 双方平等（化形妖修 / 势力盟约） - 灵魂契约 — 本命绑定（灵兽常用，一荣俱荣） - 魂契 — 灵魂碎片融合（禁忌，极端绑定）

    大乘之上 → 飞升 / 仙界（v2.0 新增）¶

    大乘期突破飞升之劫，进入仙界的新体系，与下界规则完全不同：

    • 飞升 — 大乘期渡飞升之劫，离开下界
    • 飞升之劫 — 九九天劫的最后 3 道（最凶险）
    • 界面壁障 — 人界/灵界/仙界之间的天然屏障
    • 仙界 — 飞升目的地（仙气浓郁 / 寿元无限）
    • 仙灵石 — 仙界通用资源（5 档）
    • 仙职 — 仙界等级制度（散仙→仙帝 8 档）
    • 仙劫 — 仙人修炼天劫（5 种）
    • 仙界势力 — 仙山 / 仙门 / 仙朝 / 仙盟
    • 仙界功法 — 仙级 + 神级功法
    • 下界飞升者 — 飞升者分类与路径

    仙帝之上 → 神界（v2.3 新增）¶

    仙帝渡神劫后跃迁至神界，与仙界、下界规则完全不同。神界以 神格 定起点、以 神职 定地位、以 神气/神石 为资源、以 神域 为势力范围：

    • 神界 — 修仙的最高界面（仙帝之上的界面）
    • 神格 — 飞神者起点（6 档，决定初始神职上限）
    • 神职 — 神界等级制度（神民→道祖 10 档）
    • 神气 — 神界灵气（6 档，对应不同神职）
    • 神石 — 神界通用资源（6 档）
    • 神劫 — 仙帝进入神界必须渡过的 6 类神劫
    • 神域 — 神界四大势力（天庭 / 九天神宫 / 万象 / 幽冥）
    • 神兽 — 神界顶级灵兽（青龙 / 白虎 / 朱雀 / 玄武 / 鲲鹏 / 烛龙）
    • 神器 — 神界顶级法器（造化玉碟 / 混沌钟 / 盘古幡 / 诛仙剑阵）
    • 神族 — 神界原住种族（龙 / 凤 / 麒麟 / 巫 / 人族神界后裔）
    • 下界飞神者 — 仙帝飞升神界者的分类（4 类：普通/天才/传说/神帝候选）
    • 神界纪元 — 神界 6 个纪元（太初 / 洪荒 / 上古 / 中古 / 近古 / 当代）

    仙界背面 → 魔界（v2.4 新增）¶

    魔界是天道法则的反面显化，与仙界互为镜像。仙人/神人入魔后堕入魔界：

    • 魔界 — 天道反面的第四界面
    • 魔气 — 魔界能量形态（6 档浓度）
    • 魔石 — 魔界通用资源（6 档，下品魔石→魔源石）
    • 魔修 — 魔界修士等级（散魔→魔帝 8 档，与仙职对称）
    • 魔功 — 魔界功法（10 部，与仙界功法对称）
    • 魔劫 — 飞升魔界的 5 类天劫（噬心 / 天魔 / 血魔 / 欲魔 / 心魔）
    • 魔族 — 魔界原住种族（罗刹 / 夜叉 / 修罗 / 阿修罗 / 魔族 / 人族后裔）
    • 魔兽 — 魔界灵兽（穷奇 / 梼杌 / 饕餮 / 九尾魔狐 / 暗麒麟 / 冥蛇）
    • 魔器 — 魔界法器（弑神枪 / 幽冥白骨幡 / 轮回镜 / 万魔塔 / 血海罗盘 / 阿鼻剑）
    • 魔域 — 魔界四大势力（幽都 / 血海 / 九幽 / 万魔殿）
    • 入魔者 — 从下界/仙界堕入魔界者（4 类）
    • 魔界纪元 — 魔界 6 纪元（鸿蒙 → 太古 → 远古 → 上古 → 中古 → 当代）

    死后世界 → 冥界（v2.4 新增）¶

    冥界是死灵的归宿，由轮回大道统御。仙凡陨落后魂魄入冥界：

    • 冥界 — 死灵界面
    • 冥气 — 死气浓度等级（6 档，极淡→死源）
    • 冥石 — 冥界通用资源（6 档，下品冥石→魂石）
    • 冥修 — 鬼修等级（游魂→冥帝 8 档）
    • 冥功 — 冥界功法（10 部鬼道/魂道/轮回道功法）
    • 冥劫 — 鬼修 4 类天劫（阴火 / 天雷阴劫 / 轮回劫 / 万魂噬心）
    • 冥族 — 冥界种族（鬼族 / 幽魂族 / 冥龙族 / 冥凤族 / 夜游神 / 判官族）
    • 冥兽 — 冥界灵兽（谛听 / 毕方冥种 / 九幽冥蛇 / 忘川冥蝶 / 黄泉冥鱼 / 三足冥鸦）
