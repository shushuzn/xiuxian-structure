# 阵法

## 定义
以阵旗、阵盘、灵石等为媒介，按特定规律布置的法术体系。
阵法是修仙界应用最广的辅助手段之一——攻防、困敌、聚灵、防御无所不包。

## 构成要素

### 类别

| 类别 | 用途 |
|---|---|
| **攻击型阵法** | 以阵法激发攻击法术，如"九霄雷阵" |
| **防御型阵法** | 形成护罩阻挡攻击，如"金光护山阵" |
| **困敌型阵法** | 困住敌人无法脱身，如"迷天幻阵" |
| **聚灵型阵法** | 聚集天地灵气加速修炼，常用于灵脉 |
| **幻术型阵法** | 制造幻境迷惑敌人 |
| **辅助型阵法** | 加速炼丹炼器、增幅法术威力 |

### 等级

| 等级 | 对应修士境界 | 布阵难度 |
|---|---|---|
| 凡阵 | 炼气期 | 低 |
| 灵阵 | 筑基期 | 中 |
| 宝阵 | 结丹期 | 高 |
| 仙阵 | 元婴期及以上 | 极难 |

### 构成要素
- **阵旗/阵盘**：布阵的物理媒介
- **阵眼**：阵法核心，通常放置灵石或法器
- **阵基**：阵法根基，决定阵法威力
- **阵图**：布阵的图纸与规律

### 布阵方式
- **临时布阵**：战前快速布置，灵活但威力有限
- **永久布阵**：宗门/家族长期布设，威力大但成本高
- **随身布阵**：刻在法器上，随身携带

### 限制
- 高阶阵法材料昂贵、布设耗时
- 阵法一旦识破规律，可被针对性破解
- 范围有限，需事先勘测地形
