# 魔兽

## 定义

魔界特有的高阶生物——实力远超仙界灵兽，部分甚至超过魔修。
魔兽是魔道的宠儿，拥有独特的天赋神通（杀戮、嗜血、吞噬）。
魔兽与魔修可签订契约（参考 [契约体系](../契约体系/契约.md)），也可被魔修收服。

## 等级

| 等级 | 实力 | 代表 |

| 凡兽 | 凡人级 | 魔界低阶生物 |
| 灵兽 | 仙人级 | 普通魔兽 |
| 魔兽 | 仙人级以上 | 高阶魔兽 |
| 圣兽 | 魔君级 | 顶级魔兽 |
| 魔兽王 | 魔王级 | 传说魔兽 |
| 道兽 | 魔帝级 | 传说级 |

## 代表魔兽

### 上古魔兽（顶级）

- **穷奇** — 魔王级，凶兽之首
- **梼杌** — 魔王级，混沌凶兽
- **饕餮** — 魔王级，吞噬一切
- **九尾魔狐** — 魔君级，幻术无双
- **暗麒麟** — 魔王级，黑暗守护
- **冥蛇** — 魔君级，剧毒无双

### 普通魔兽

- **血蝙蝠** — 魔使级，群居
- **幽冥狼** — 魔官级，群居
- **噬魂蝶** — 魔将级，吸取神魂

## 魔兽与人

### 签约

- 魔兽可与魔修签订契约（参考 [契约体系](../契约体系/契约.md)）
- 多采用「血契」或「魂契」
- 「主从契约」常见（魔兽比神兽更易臣服于强者）

### 化形

- 魔兽可化为人形
- 化形后保留部分魔兽特征（如暗麒麟角）
- 化形魔兽可参与魔界政治

### 战斗

- 魔兽战斗力极强
- 部分魔兽可与魔王单挑
- 顶级魔兽需魔帝出手

## 关键剧情节点

- 穷奇是魔界四大凶兽之首
- 九尾魔狐是罗刹族的守护魔兽
- 暗麒麟是魔界正统的象征

## 备注

- 魔兽比神兽更凶残
- 魔兽驯服率比神兽低（多采用血契）
- 饕餮是传说级，吞噬万物
